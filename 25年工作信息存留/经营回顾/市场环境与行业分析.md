# 市场环境与行业分析

> [!abstract] 概述
> 2025年全球化工及罐箱市场面临结构性调整与周期性低迷的双重压力，本文提炼自总裁报告中的宏观分析部分。

## 宏观经济环境

### 地缘政治与关税

- 美国对华关税政策反复，2025年8月延期90天实施部分关税
- 关键领域（稀土/半导体/精细化工品）持续博弈
- 全球干散货贸易剧烈波动
- ==企业投资意愿明显降温==，产能扩张趋于谨慎

### PMI指标

- 中国以外国家PMI显著下降
- 多数欧美国家PMI相对疫情前处于低位
- 反映世界经济增量动能持续萎缩

## 化工行业态势

### 全球化工产业

| 特征 | 表现 |
|------|------|
| 欧洲 | 大量化工企业濒临倒闭，产出增长率仅0.5% |
| 中国 | 化工产能继续增加，价格持续下行 |
| 中东 | 新产能逐渐上升 |
| 利润 | 仍处于低谷，蒸汽裂解装置关闭 |

### 国内化工市场

- 价格持续下行，行业竞争加剧
- 陷入非理性=="内卷"==状态
- 成本压力沿产业链向下传导
- 运输运营商利润压缩 → 采购能力下降
- 槽罐车运力回流（以牺牲安全为代价的临时策略）

## 罐箱市场分析

### 供需格局

| 维度 | 数据 |
|------|------|
| 全球供给端产能 | ~==12万台== |
| 实际产量 | ~==6万台== |
| 产能利用率 | ~50% |

### 客户行为变化

| 客户类型 | 行为 |
|----------|------|
| **大运营商** | 基本==停止采购==，观望状态 |
| **租赁商** | 资金成本高，投资意愿不强；停止"囤箱待涨"，消化库存 |
| **新兴运营商** | 需求增加但总量有限，价格苛刻、风险大 |

### 价格与竞争

- 标罐25T11/26T11价格持续低位
- 2025年H1标罐单价同比↓11.9%
- 竞争对手低价抢单
- 运营商占比下降13个百分点，租赁商占比上升15个百分点
- 产品结构变化：标罐占比提升6个百分点，特罐/碳罐下降

### 医疗市场环境

**机遇：**
- 核心客户处于产品变革期
- 英国西门子新工厂扩张需求

**挑战：**
- 美国关税影响MR设备出口
- 国内新晋者冲击集采价格
- 低端产品价格战突出

**策略：** 突破MR单一产品，拓展全系列医疗影像设备

## 对26年的启示

> [!tip] 关键洞察
> 1. 化工周期低迷短期难以逆转，26年罐箱仍将承压
> 2. 大运营商恢复采购是量能回升的关键信号——26年Q1已见3月暴增4,552台
> 3. 产能过剩（全球12万 vs 实际6万）意味着价格竞争持续
> 4. 医疗和新赛道是穿越周期的核心抓手
> 5. 国内化工"内卷"趋势下，终端客户合作（万华/红宝丽）模式值得扩大
