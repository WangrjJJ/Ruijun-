# 数字化与智能制造成果

> [!abstract] 概述
> 2025年数字化转型围绕"降本增效"主线，在RPA、BI、自动化产线、5G工厂等方面取得实质成果，为26年深化数字化提供基础。

## RPA流程自动化

| 维度 | 数据 |
|------|------|
| 已上线流程 | ==20+个== |
| 平均效率提升 | ==+45%== |
| 覆盖范围 | 财务对账、采购单据、报关单证、HR考勤等 |

> [!tip] 关键突破
> RPA从试点走向规模化应用，证明了"小切口、快迭代"的推广策略有效。

## BI与数据分析

### 订单级毛利测算

- 实现==单箱级别==的毛利实时测算
- 涵盖材料、人工、制造费用、管理费用分摊
- 为报价决策提供数据支撑（标罐毛利仅5%时尤为关键）

### 固定资产数据分析

- 完成全公司固定资产台账数字化
- 资产利用率分析支撑产能优化决策
- 为26年产能利用率提升提供基线数据

### BI看板体系

- 经营驾驶舱覆盖营收/利润/订单/产能核心指标
- 支持从公司→业务线→产品线的逐级下钻
- 数据刷新频率从月度提升至==周度==

## 智能制造与自动化

### 防波板自动折弯系统

| 维度 | 数据 |
|------|------|
| 效率提升 | ==+400%== |
| 状态 | 首台验收，批量投产 |
| 意义 | V型防波板从手工折弯转自动化，质量一致性大幅提升 |

### 其他自动化改造

| 项目 | 成果 |
|------|------|
| 喷粉车间 | VOCs超低排放 + 每台节约346元，累计节约超1,000万 |
| 有色金属精密制造中心 | 激光切割/折弯/剪板/冲床投用，箱类外协加工件全部转自制 |
| 焊接自动化 | 部分工位实现机器人焊接，焊缝质量稳定性提升 |

### 5G全连接工厂

- ==获得工信部5G全连接工厂认证==
- 5G网络覆盖生产核心区域
- 支撑AGV调度、视觉检测、远程运维等场景
- 为后续数字孪生、AI质检奠定网络基础

## 数字化关务合规

- 关务系统与ERP/报关行系统打通
- 单证流转效率提升，人工差错率下降
- 归类准确率提升，降低合规风险

## 25年 → 26年能力传递

| 能力 | 25年基础 | 26年深化方向 |
|------|----------|-------------|
| RPA | 20+流程上线 | 扩展至供应链/质量领域 |
| BI | 订单级毛利测算 | ==实时成本预警== |
| 自动化 | 防波板+喷粉+精密中心 | 更多工位机器人替代 |
| 5G工厂 | 认证通过 | 数字孪生试点 |
| 数据基础 | 资产/关务数字化 | ==全面数据治理== |

## 关键决策点

> [!warning] 26年数字化深化的挑战
> 1. 数字化投入ROI需更精准量化（25年部分项目缺乏效益闭环）
> 2. 数据孤岛仍存在——ERP/MES/WMS/BI之间的数据一致性待解决
> 3. 产能利用率仅38%（26年Q1），自动化产线的价值受限于开工率
> 4. 人才储备：数字化复合型人才缺口仍大
