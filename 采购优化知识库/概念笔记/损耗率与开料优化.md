# 损耗率与开料优化

> [!abstract] 定义
> 在钢板采购场景中，原材料板材宽度与产品需求宽度之间的差值产生的**开料废边**，称为 Trim Loss。这是[[钢材采购优化模型]]中影响成本的关键因素。

## 损耗率公式

$$
\text{trim\_loss} = \frac{W_{\text{material}} - W_{\text{product}}}{W_{\text{material}}}
$$

$$
\text{yield\_rate} = 1 - \text{trim\_loss}
$$

> [!example] 计算示例
> - 标准板材 1500mm 切割 1200mm 产品：$(1500-1200)/1500 = 20\%$ 损耗
> - 非标板材 1219mm 切割 1200mm 产品：$(1219-1200)/1219 = 1.6\%$ 损耗
> - 定尺板材 1200mm 切割 1200mm 产品：$0\%$ 损耗

## 标准 vs 非标的经济性抉择

| 维度 | 标准材料 | 非标材料 |
|------|----------|----------|
| 单价 | 较低（¥/吨） | ==较高（+溢价）== |
| 损耗率 | 高（10%~20%） | ==极低（0%~2%）== |
| MOQ | 低/无 | ==较高== |
| 交期 | 短（现货） | 较长（1~3周） |
| 实际综合成本 | 单价低但浪费多 | 单价高但浪费少 |

> [!tip] 核心洞察
> ==并非单价越低越好==。当损耗率超过非标溢价比例时，采购非标材料反而更经济。这正是优化模型自动求解的价值所在。

## 在模型中的实现

```python
# solve_steel.py 中的损耗计算
def calc_trim_loss(material_width, product_width):
    if material_width < product_width - 0.1:
        return None  # 不可用
    trim = (material_width - product_width) / material_width
    return trim

# 兼容性判断：宽度 + 厚度 + 品质等级
def is_compatible(material, product):
    return (material.width >= product.width - 0.1
            and abs(material.thickness - product.thickness) <= 0.005
            and material.grade_rank >= product.min_grade_rank)
```

## 钢材品质等级层级

```
2B（标准面）→ BA（光亮退火）→ #4（拉丝）→ HL（发线）→ Mirror（镜面）
     低 ─────────────────────────────────────────── 高
```

> [!warning] 向上兼容规则
> 高品质等级材料可以替代低品质需求（如用 BA 代替 2B），但**不可反向**。模型中通过 `grade_rank` 数值约束实现。
