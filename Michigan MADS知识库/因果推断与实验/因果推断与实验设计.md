## 因果推断部分

## 核心知识：潜在结果框架 (Potential Outcomes)

### Rubin 因果模型

对每个个体 $i$，定义两个==潜在结果==：

| 符号 | 含义 |
|------|------|
| $Y_i^1$ | 接受处理时的结果 |
| $Y_i^0$ | 未接受处理时的结果 |
| $D_i$ | 处理变量（0 或 1） |

==根本问题==：对同一个体，我们只能观察到一个潜在结果，另一个是==反事实== (counterfactual)。

> [!example] S630 简历实验
> Bertrand & Mullainathan (2004) 实验中：$D_i$ = 是否使用黑人名字（`black`），$Y_i$ = 是否收到回电（`call`）。对 id=345 的观测，`black=1`，因此反事实结果为 $Y_{345}^{white}$（若使用白人名字会怎样）。

### 平均处理效应 (ATE)

$$\text{ATE} = E[Y_i^1 - Y_i^0] = E[Y_i^1] - E[Y_i^0]$$

$$\text{ATT} = E[Y_i^1 - Y_i^0 | D_i = 1]$$

### 协变量平衡检验

在估计因果效应前，必须检查处理组和控制组的==协变量分布是否相似==：

```python
from scipy import stats
# 两组均值差异的 t 检验
ttest = stats.ttest_ind(
    data[var][data['treatment']==1],
    data[var][data['treatment']==0])
```

> [!tip] 为什么协变量平衡至关重要
> S630 观察数据中，黑人与白人在 `somecol_more`（$p=3.75 \times 10^{-29}$）和 `yearsexp`（$p=4.74 \times 10^{-93}$）上存在==显著差异==。因此即使就业率有种族差距，也==不能==从观察数据中推断种族歧视——差异可能来自教育和经验的不同。

### 随机实验的优势

| 比较项 | 观察数据 | 随机实验 |
|--------|----------|----------|
| 协变量平衡 | 通常不平衡 | 随机化保证平衡 |
| 混杂因素 | 可能存在 | 被随机化消除 |
| ATE 估计 | 有偏 | 无偏 |
| 因果推断 | 困难 | 直接 |

> [!example] S630 随机实验结论
> 随机实验中，各协变量（`male`, `computerskills`, `education`, `ofjobs`, `yearsexp`）在 `black=0/1` 两组间分布相似。回电率 t 检验 $p = 3.94 \times 10^{-5}$，可以==因果性地==推断存在针对黑人名字的歧视。女性子组 $p = 2.80 \times 10^{-4}$ 显著，男性子组 $p = 0.051$ 边际不显著。

## 核心知识：匹配方法

### 最近邻匹配

为每个处理组个体找到协变量最相似的控制组个体，计算配对差异：

```python
from causalinference import CausalModel
cm = CausalModel(Y=y.values, D=d.values, X=X.values)
cm.est_via_matching()
ate = cm.estimates['matching']['ate']
```

> [!example] Nike Vaporfly 跑鞋实验
> S630 中直接比较穿/不穿 Vaporfly 的均值差异为 -0.064（6.4% 更快），但仅按 `age` 匹配后 ATE = ==-0.037==（3.7%），加入多协变量倾向得分匹配后 ATE = ==-0.037==。控制混杂因素后效应明显缩小。

### 倾向得分匹配 (Propensity Score Matching)

倾向得分 $e(x) = P(D=1|X=x)$，用 logit 模型估计：

```python
cm = CausalModel(Y=y, D=d, X=X[['age','male','seasoned','enthusiastic']])
cm.est_propensity_s()
cm.trim_s()  # 裁剪极端倾向得分
cm.est_via_matching()
```

## 核心知识：控制回归与遗漏变量偏差

### OLS 控制回归

回归是==自动化的匹配器==（Mastering Metrics）：

$$\ln(\text{race\_time}) = \beta_0 + \beta_1 \text{vaporfly} + \beta_2 \text{male} + \beta_3 \text{seasoned} + \beta_4 \text{enthusiastic} + \epsilon$$

```python
import statsmodels.api as sm
X = sm.add_constant(data[['vaporfly','male','seasoned','enthusiastic']])
model = sm.OLS(y, X).fit()
robust = model.get_robustcov_results()
```

### 遗漏变量偏差 (OVB)

$$\text{OVB} = (\text{遗漏变量对结果的效应}) \times (\text{处理变量对遗漏变量的效应})$$

> [!example] OVB 实证
> S630 中遗漏 `age` 时，`vaporfly` 系数为 ==-0.066==；加入 `age` 控制后变为 ==-0.043==。OVB 的方向验证了直觉分析：年长者更可能买得起 Vaporfly（正），年长者跑得更快（负），OVB = 正 $\times$ 负 = ==负==，导致遗漏 `age` 时高估了鞋的效果。

## 实验设计部分

## 核心知识：实验设计原则

### 实验类型

| 类型 | 特点 | 内部效度 | 外部效度 |
|------|------|----------|----------|
| 实验室实验 | 高控制 | 高 | 低 |
| 现场实验 | 自然环境 | 中-高 | 高 |
| 在线实验 (A/B) | 大样本、快速迭代 | 中 | 中 |

### 随机化检验

验证两组是否正确随机分配：

```python
def objective_randomization(data):
    for var in ['female', 'age', 'hispanic']:
        g1 = data[data['treatment']=='A'][var]
        g2 = data[data['treatment']=='B'][var]
        t_stat, p_val = stats.ttest_ind(g1, g2)
```

> [!example] S631 拍卖实验
> S631 中对 `k1_8_exp_lot` 和 `k1_8_lot_exp` 两个处理组检验随机化。`female` 的 $t=-0.23$, $p=0.82$；`age` 的 $t=-1.86$, $p=0.07$；`hispanic` 的 $t=-0.88$, $p=0.38$。所有 $p>0.05$，随机化==有效==。

def stats_calculator(data):

## 核心知识：假设检验

### t 检验类型

| 类型 | 适用场景 | scipy 函数 |
|------|----------|-----------|
| 独立双样本 t 检验 | 两组均值比较 | `stats.ttest_ind()` |
| 配对 t 检验 | 同一对象前后比较 | `stats.ttest_rel()` |
| 单样本 t 检验 | 均值与常数比较 | `stats.ttest_1samp()` |

### 决策规则

给定显著性水平 $\alpha=0.05$（双侧）：
- $p < 0.025$（单侧）或 $p < 0.05$（双侧） $\Rightarrow$ 拒绝 $H_0$
- $|t| > t_{crit}$ $\Rightarrow$ 拒绝 $H_0$

> [!tip] 双侧 vs 单侧
> S630 中所有 t 检验均为双侧。双侧检验的判断标准为 $p < \alpha$（如 $p < 0.05$），而非 $p < \alpha/2$。`scipy.stats.ttest_ind` 默认返回双侧 p 值。

## 核心知识：A/B 测试与效力分析

### A/B 测试框架

1. **假设设定**：$H_0$: 处理无效（均值相等）；$H_1$: 处理有效
2. **样本量计算**：基于效力分析确定所需样本
3. **随机分配**：将用户随机分入控制/处理组
4. **数据收集**：运行足够时间
5. **统计检验**：t 检验或 z 检验

### 效力分析 (Power Analysis)

$$n = \left(\frac{z_{1-\alpha/2} + z_{1-\beta}}{\delta / \sigma}\right)^2$$

| 参数 | 含义 | 典型值 |
|------|------|--------|
| $\alpha$ | 一类错误率 | 0.05 |
| $\beta$ | 二类错误率 | 0.20 |
| $1-\beta$ | 统计功效 | 0.80 |
| $\delta$ | 最小可检测效应 | 由业务决定 |
| $\sigma$ | 标准差 | 从历史数据估计 |

### ANOVA 方差分析

当处理组 $\geq 3$ 时，使用 ANOVA 而非多个 t 检验：

$$F = \frac{\text{MSB}}{\text{MSW}} = \frac{\text{组间方差}}{\text{组内方差}}$$

## 方法选择指南

| 场景 | 推荐方法 | 理由 |
|------|----------|------|
| 可做实验 | 随机对照实验 (RCT) | 金标准，无混杂 |
| 不可实验 + 有工具变量 | IV 回归 | 处理内生性 |
| 不可实验 + 断点存在 | 断点回归 (RDD) | 局部随机化 |
| 不可实验 + 前后数据 | 双重差分 (DID) | 控制时间趋势 |
| 不可实验 + 可观测混杂 | 倾向得分匹配 / OLS | 条件独立假设 |
| 多处理组比较 | ANOVA | 控制多重比较 |
| 在线产品优化 | A/B Testing | 大样本、快速 |

## 实战洞察

- ==观察数据中的相关性不等于因果性==：S630 观察数据中就业率的种族差异可能来自教育/经验差异而非歧视
- 随机实验是因果推断的==金标准==，但实验中仍需检验随机化是否有效（balance check）
- OVB 公式是评估遗漏变量影响方向的==有力直觉工具==，即使无法精确计算
- 使用 `ln(outcome)` 可将系数直接解释为==百分比变化==（如 -0.043 即 4.3% 更快）
- 倾向得分匹配前需要 `trim_s()` 裁剪极端值，否则匹配质量差
- A/B 测试中==样本量不足==是最常见的错误——效力分析应在实验前进行
- 多重比较问题（如同时检验 20 个指标）需要 Bonferroni 校正或 FDR 控制
- S631 中实验室拍卖实验的 order effect（先做彩票还是先做拍卖）需作为处理变量分析
