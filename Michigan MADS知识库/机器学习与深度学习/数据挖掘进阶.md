## 核心知识：N-gram 语言模型

### 基本概念

N-gram 是连续 $n$ 个 token 的序列。N-gram 语言模型假设当前 token 的概率仅取决于前 $n-1$ 个 token（==马尔可夫假设==）：

$$P(w_t | w_1, \ldots, w_{t-1}) \approx P(w_t | w_{t-n+1}, \ldots, w_{t-1})$$

### 构建流程

1. **加载与分词**：读取文本，逐句分词（`nltk.word_tokenize`）
2. **构建词表**：收集唯一 token，添加 `<s>`（句首）和 `</s>`（句尾）
3. **构建 N-gram**：对每个句子添加 $n-1$ 个 `<s>` 前缀和 `</s>` 后缀
4. **训练模型**：使用 `nltk.lm.MLE` 最大似然估计

```python
from nltk.lm import MLE
lm = MLE(n)
lm.fit(ngram_list, vocabulary_text=vocab)
# 生成文本
line = lm.generate(num_words, text_seed=["<s>"]*(n-1))
```

> [!example] 莎士比亚十四行诗生成
> S632 中使用莎士比亚十四行诗训练 3-gram 模型，生成的诗句如 "because i would be forgot, for all that is my love still telling"。随着 $n$ 增大，生成文本更接近原文风格但多样性降低。

### Bigram 的马尔可夫性质

> [!tip] 关键理解
> Bigram 是==一阶马尔可夫链==：无论给定多长的 `<s> <s> <s>` 前缀，bigram 模型只看==最后一个== token 来决定下一个 token。这意味着对 bigram 而言，$P(w|<s>,<s>,<s>) = P(w|<s>)$。

## 核心知识：隐马尔可夫模型 (HMM)

### 核心组件

| 组件 | 符号 | 含义 |
|------|------|------|
| 隐状态集 | $S$ | 如 POS 标签集 {NN, VB, JJ, ...} |
| 观测集 | $O$ | 如单词集 |
| 转移概率 | $A = \{a_{ij}\}$ | $P(s_t = j | s_{t-1} = i)$ |
| 发射概率 | $B = \{b_j(o)\}$ | $P(o_t | s_t = j)$ |
| 初始概率 | $\pi$ | $P(s_1 = i)$ |

### HMM 三大问题

| 问题 | 描述 | 算法 |
|------|------|------|
| 评估 | 给定模型和观测序列，求概率 | 前向/后向算法 |
| 解码 | 给定模型和观测序列，求最优隐状态 | ==Viterbi 算法== |
| 学习 | 给定观测序列，估计模型参数 | Baum-Welch (EM) |

```python
from nltk.tag.hmm import HiddenMarkovModelTagger
hmm = HiddenMarkovModelTagger.train(labeled_sentences)
tagged = hmm.tag(["The", "cat", "sat"])
# 评估
hmm.test(dev_set)  # 输出 accuracy
```

## 核心知识：时间序列模式

### 季节分解

加法模型将时间序列分解为三个成分：

$$Y(t) = T(t) + S(t) + R(t)$$

| 成分 | 含义 | 特征 |
|------|------|------|
| $T(t)$ | 趋势 | 长期变化方向 |
| $S(t)$ | 季节性 | 周期重复模式 |
| $R(t)$ | 残差 | 随机波动 |

```python
from statsmodels.tsa.seasonal import seasonal_decompose
result = seasonal_decompose(series, model="additive")
# result.trend, result.seasonal, result.resid
```

### 趋势拟合

使用多项式回归拟合趋势曲线：

```python
from sklearn.preprocessing import PolynomialFeatures
train_X = PolynomialFeatures(10).fit_transform(
    np.arange(len(series)).reshape(-1, 1))
lin_reg = LinearRegression().fit(train_X, series.values)
```

> [!tip] S632 实验
> 对全球 COVID-19 每日新增确诊数使用 10 阶多项式拟合趋势，$R^2 \geq 0.95$。$x_i$ 使用==整数时间索引== `np.arange(len(series))`，$y_i$ 为每日新增数。

### 移动平均

**加权移动平均 (WMA)**：

$$x_j' = \frac{\sum_{i=1}^{k} w_i \cdot x_{j-k+i}}{\sum_{i=1}^{k} w_i}$$

**时间指数移动平均 (Time EMA)**：

$$x_j' = \frac{\sum_{i=1}^{j} \exp[-\lambda(t_j - t_i)] \cdot x_i}{\sum_{i=1}^{j} \exp[-\lambda(t_j - t_i)]}$$

当 $\lambda=0$ 时退化为==累积移动平均 (CMA)==。$\lambda$ 越大，近期数据权重越高。

```python
# WMA
series.rolling(window).apply(
    lambda x: np.sum(x * weights) / np.sum(weights))
# EMA (statsmodels)
from statsmodels.tsa.api import SimpleExpSmoothing
ema = SimpleExpSmoothing(series).fit(smoothing_level=0.5)
```

## 核心知识：时间序列相似性

### 三种距离度量

| 度量 | 公式 | 特点 |
|------|------|------|
| 欧氏距离 | $\sqrt{\sum(x_i - y_i)^2}$ | 逐点对齐，对时间偏移敏感 |
| 余弦相似度 | $\frac{x^T y}{\|x\|_2 \|y\|_2}$ | 关注形状而非幅度 |
| DTW 代价 | 动态规划最优对齐 | 允许时间弹性伸缩 |

### 动态时间规整 (DTW)

DTW 通过动态规划找到两个序列之间的==最优对齐路径==：

$$\text{DTW}(i,j) = d(x_i, y_j) + \min \begin{cases} \text{DTW}(i, j-1) \\ \text{DTW}(i-1, j) \\ \text{DTW}(i-1, j-1) \end{cases}$$

其中 $d(x_i, y_j) = (x_i - y_j)^2$。

```python
def dtw_cost(x, y):
    cost = np.zeros((len(y), len(x)))
    cost[0, 0] = (y[0] - x[0]) ** 2
    for i in range(1, len(y)):
        cost[i, 0] = (y[i]-x[0])**2 + cost[i-1, 0]
    for j in range(1, len(x)):
        cost[0, j] = (y[0]-x[j])**2 + cost[0, j-1]
    for i in range(1, len(y)):
        for j in range(1, len(x)):
            cost[i,j] = (y[i]-x[j])**2 + min(
                cost[i-1,j], cost[i,j-1], cost[i-1,j-1])
    return cost[-1, -1]
```

> [!example] COVID-19 五国季节性模式对比
> S632 中对 US/Brazil/India/Russia/South Africa 的每日新增季节性成分计算相似度。结论：India 与 South Africa 季节模式==最相似==（欧氏距离 4490，DTW 4463），Brazil 与其他国家差异最大。DTW 距离通常 $\leq$ 欧氏距离，因为允许时间弹性对齐。

## 方法选择指南

| 场景 | 推荐方法 | 理由 |
|------|----------|------|
| 文本生成/语言模型 | N-gram | 简单高效，可解释 |
| 序列标注 (POS/NER) | HMM / CRF | 建模隐状态序列 |
| 趋势发现 | 季节分解 + 多项式回归 | 分离趋势/季节/残差 |
| 平滑噪声 | WMA / EMA | 参数化平滑 |
| 不等长序列相似度 | DTW | 弹性时间对齐 |
| 等长序列形状比较 | 余弦相似度 | 忽略幅度差异 |
| 等长序列距离 | 欧氏距离 | 直观、计算简单 |

## 实战洞察

- N-gram 的 $n$ 越大，生成越"像"训练数据但多样性越低；$n=3$ 通常是文本生成的好起点
- HMM 训练数据中只保留长句（$\geq 10$ tokens），短句可能导致参数估计不稳定
- COVID-19 数据有明显的==周末效应==（7天周期），`seasonal_decompose` 默认 `period=7` 可直接捕获
- DTW 的时间复杂度为 $O(nm)$，大规模时序需考虑下界剪枝或近似 DTW
- 比较时序相似性时，先提取季节成分再度量，比直接度量原始序列更有意义
- WMA 的权重设计（线性、指数等）会显著影响平滑结果
