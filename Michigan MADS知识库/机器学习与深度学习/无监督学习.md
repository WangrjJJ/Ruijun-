## 核心知识：主成分分析 (PCA)

### 数学基础

PCA 通过==奇异值分解 (SVD)== 找到数据方差最大的方向：

$$X^* = U \Sigma V^T$$

其中主成分方差为：

$$\text{Var}(PC_k) = \frac{\sigma_k^2}{n - 1}$$

### PCA 工作流程

1. 数据标准化（`StandardScaler`，使均值为0、标准差为1）
2. 拟合 PCA 模型
3. 解读主成分载荷（`components_`）
4. 计算方差解释比（`explained_variance_ratio_`）

```python
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA
X_norm = StandardScaler().fit_transform(X)
pca = PCA(n_components=3, random_state=0).fit(X_norm)
```

> [!example] 啤酒评分数据集 PCA
> S543 Assignment 1 对啤酒评分（appearance/aroma/palate/taste/overall）做 PCA，3 个主成分方差分别为 ==3.835, 0.486, 0.311==。第一主成分占总方差的 76.7%，所有评分维度在 PC1 上载荷接近（-0.40 ~ -0.48），说明 PC1 代表"整体品质"。

### Biplot 解读

Biplot 同时展示数据点和变量向量，关键解读规则：

| 观察 | 含义 |
|------|------|
| 向量方向相近 | 变量高度正相关 |
| 向量方向相反 | 变量负相关 |
| 向量正交 | 变量接近零相关 |
| 向量长度 | 变量对该 PC 的贡献大小 |

> [!tip] S543 实验结论
> `review_palate` 与 PC1 相关性最高；`review_appearance` 与 PC2 相关性最高；`review_appearance` 与 `review_overall` 接近零相关——外观好不等于整体体验好。

## 核心知识：流形学习

### 方法对比

| 方法 | 保持全局结构 | 保持局部结构 | 保持密度 | 速度 |
|------|:----------:|:----------:|:-------:|:----:|
| PCA | 是 | 一般 | 一般 | 快 |
| t-SNE | 否 | 是 | 否 | 慢 |
| MDS | 是 | 一般 | 一般 | 中 |
| UMAP | 可调 | 是 | 部分 | 快 |

```python
from sklearn.manifold import TSNE, MDS
X_tsne = TSNE(random_state=42).fit_transform(X)
X_mds = MDS(random_state=42).fit_transform(X)
```

> [!tip] S543 实验结论
> 对含多种形状聚类的高维数据（9维多项式特征）：t-SNE ==不==保持聚类密度，==不==保持簇间相对距离；MDS ==保持==簇间相对距离。选择方法需根据分析目的。

## 核心知识：密度估计与异常检测

### 核密度估计 (KDE)

非参数密度估计，每个数据点放置一个核函数（通常高斯）：

$$\hat{f}(x) = \frac{1}{nh} \sum_{i=1}^{n} K\left(\frac{x - x_i}{h}\right)$$

其中 $h$ 为==带宽==参数，控制平滑程度。

```python
from sklearn.neighbors import KernelDensity
kde = KernelDensity(kernel='gaussian', bandwidth=5)
kde.fit(X_train)
log_density = kde.score_samples(X_test)
```

> [!example] 基于 KDE 的朴素贝叶斯分类
> S543 中用 KDE 估计心脏病 (CHD) 和非 CHD 人群的收缩压分布，再通过贝叶斯公式分类：
> $$\log p(c|x) \propto \log p(x|c) + \log p(c)$$
> 先验 $p(c_{CHD})=0.5$，测试集准确率达 0.70。

### 局部异常因子 (LOF)

LOF 通过比较数据点与其邻居的==局部密度偏差==识别异常：

```python
from sklearn.neighbors import LocalOutlierFactor
clf = LocalOutlierFactor(n_neighbors=10)
clf.fit(X_umap_embedding)
scores = clf.negative_outlier_factor_
# 值越负表示越异常
outliers = scores.argsort()[:20]
```

> [!tip] UMAP + LOF 组合
> 先用 UMAP 降至 2 维（降低 LOF 的维度诅咒），再用 LOF 检测异常。S543 中啤酒数据集异常点具有极端评分特征（如 appearance 极高但 overall 极低）。

## 核心知识：聚类算法

### K-Means

$$\min_{\mu} \sum_{k=1}^{K} \sum_{x_i \in C_k} \|x_i - \mu_k\|^2$$

- 需预设 $K$ 值
- 对初始化敏感（`n_init=10` 默认多次运行）
- 使用==轮廓系数==评估聚类质量：$s(i) = \frac{b(i) - a(i)}{\max(a(i), b(i))}$

### 层次聚类

- 凝聚法 (Agglomerative)：自底向上合并
- 连接方式：Ward（最小化方差增量）、Complete、Average、Single
- 通过树状图 (Dendrogram) 决定最优 $K$

### 聚类方法对比

| 方法 | 优势 | 劣势 |
|------|------|------|
| K-Means | 快速、可扩展 | 需预设 K，假设球形簇 |
| DBSCAN | 自动发现簇数、处理噪声 | 密度不均时困难 |
| Agglomerative | 层次结构、灵活连接 | 计算量大 O(n^3) |
| Spectral | 处理非凸形状 | 需预设 K，大数据慢 |

## 方法选择指南

| 场景 | 推荐方法 | 理由 |
|------|----------|------|
| 特征理解/降维 | PCA | 线性降维，可解释 |
| 高维可视化 | t-SNE / UMAP | 保持局部邻域结构 |
| 密度估计 | KDE | 非参数，灵活 |
| 异常检测 | UMAP + LOF | 降维后局部密度比较 |
| 球形簇聚类 | K-Means | 快速收敛 |
| 非凸形状聚类 | DBSCAN / Spectral | 不假设簇形状 |
| 层次关系发现 | Agglomerative | 树状图直觉 |

## 实战洞察

- PCA 前==必须标准化==，否则方差大的特征会主导主成分方向
- PCA `explained_variance_` 的方差值等于 $\frac{\sigma_k^2}{n-1}$，可从 SVD 的奇异值直接计算
- t-SNE 不保持全局距离，因此==不能==根据 t-SNE 图判断簇间远近
- UMAP 的 `set_op_mix_ratio` 参数控制全局/局部信息权重，异常检测时应偏向全局（取小值如 0.15）
- LOF 的 `n_neighbors` 应大于最小簇的样本数，小于最大近邻样本数；默认 20 通常够用
- KDE 带宽选择至关重要：太小过拟合（锯齿状），太大过平滑（丢失细节）
- Biplot 是同时理解样本分布和变量相关性的最佳工具
