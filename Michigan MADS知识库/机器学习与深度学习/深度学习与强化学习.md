## 深度学习部分

## 核心知识：神经网络基础

### 前馈神经网络 (MLP)

单个神经元的计算：

$$z = \sum_{i=1}^{n} w_i x_i + b, \quad a = \sigma(z)$$

常用激活函数：

| 函数 | 公式 | 特点 |
|------|------|------|
| Sigmoid | $\sigma(z) = \frac{1}{1+e^{-z}}$ | 输出 (0,1)，梯度消失 |
| ReLU | $f(z) = \max(0, z)$ | 计算快，缓解梯度消失 |
| Tanh | $\tanh(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}}$ | 输出 (-1,1)，零中心 |
| Softmax | $\frac{e^{z_i}}{\sum_j e^{z_j}}$ | 多分类输出层 |

### 反向传播 (Backpropagation)

基于==链式法则==逐层计算梯度：

$$\frac{\partial L}{\partial w^{(l)}} = \frac{\partial L}{\partial a^{(l)}} \cdot \frac{\partial a^{(l)}}{\partial z^{(l)}} \cdot \frac{\partial z^{(l)}}{\partial w^{(l)}}$$

梯度下降更新规则：

$$w \leftarrow w - \eta \frac{\partial L}{\partial w}$$

> [!tip] 学习率选择
> 学习率 $\eta$ 太大导致震荡甚至发散，太小收敛极慢。实践中常用 Adam 优化器自适应调整学习率。

### 损失函数

| 任务 | 损失函数 | 公式 |
|------|----------|------|
| 回归 | MSE | $\frac{1}{n}\sum(y_i - \hat{y}_i)^2$ |
| 二分类 | Binary Cross-Entropy | $-\sum[y\log\hat{y} + (1-y)\log(1-\hat{y})]$ |
| 多分类 | Categorical Cross-Entropy | $-\sum y_k \log \hat{y}_k$ |

## 核心知识：卷积神经网络 (CNN)

### 核心层

| 层类型 | 作用 | 关键参数 |
|--------|------|----------|
| Conv2D | 局部特征提取 | `filters`, `kernel_size`, `stride` |
| MaxPooling2D | 降采样，平移不变性 | `pool_size` |
| Flatten | 展平为一维向量 | -- |
| Dense | 全连接分类 | `units` |
| Dropout | 正则化，随机置零 | `rate` |

### 卷积输出尺寸

$$\text{output\_size} = \frac{n - k + 2p}{s} + 1$$

其中 $n$ 为输入尺寸，$k$ 为核大小，$p$ 为 padding，$s$ 为 stride。

> [!tip] CNN 设计直觉
> 浅层卷积捕获==边缘、纹理==等低级特征，深层捕获==语义、物体部件==等高级特征。增加 `filters` 数量可学习更多特征图。

## 核心知识：循环神经网络 (RNN)

### 标准 RNN

$$h_t = \tanh(W_{xh} x_t + W_{hh} h_{t-1} + b_h)$$

$$y_t = W_{hy} h_t + b_y$$

- 问题：长序列中==梯度消失/爆炸==

### LSTM

通过门控机制解决长程依赖：

| 门 | 公式 | 功能 |
|----|------|------|
| 遗忘门 | $f_t = \sigma(W_f [h_{t-1}, x_t] + b_f)$ | 决定丢弃多少旧记忆 |
| 输入门 | $i_t = \sigma(W_i [h_{t-1}, x_t] + b_i)$ | 决定写入多少新信息 |
| 输出门 | $o_t = \sigma(W_o [h_{t-1}, x_t] + b_o)$ | 决定输出多少当前记忆 |

### 正则化技术

| 方法 | 机制 | 适用场景 |
|------|------|----------|
| Dropout | 训练时随机关闭神经元 | 过拟合防止 |
| Batch Normalization | 层间标准化激活值 | 加速收敛 |
| Early Stopping | 验证集损失不降时停止 | 通用正则化 |
| Weight Decay | L2 惩罚权重 | 抑制大权重 |

## 强化学习部分

## 核心知识：马尔可夫决策过程 (MDP)

### MDP 五元组

$$\langle S, A, P, R, \gamma \rangle$$

| 元素 | 含义 |
|------|------|
| $S$ | 状态空间 |
| $A$ | 动作空间 |
| $P(s'|s,a)$ | 状态转移概率 |
| $R(s,a)$ | 即时奖励 |
| $\gamma \in [0,1)$ | 折扣因子 |

### Bellman 方程

==最优值函数==满足的递归关系：

$$V^*(s) = \max_{a} \mathbb{E}\left[r(s,a) + \gamma V^*(s')\right]$$

$$Q^*(s,a) = R(s,a) + \gamma \sum_{s'} P(s'|s,a) \max_{a'} Q^*(s', a')$$

> [!example] S644 Gridworld Maze
> 在 7 个状态的网格迷宫中，$s=6$ 为目标状态 $V^*(6) = 0$。其他状态通过 Bellman 方程迭代求解。折扣因子 $\gamma$ 控制对未来奖励的重视程度。

### $\epsilon$-贪心策略

$$a = \begin{cases} \arg\max_a Q(s,a) & \text{以概率 } 1-\epsilon \\ \text{random action} & \text{以概率 } \epsilon \end{cases}$$

> [!tip] 探索与利用的权衡
> $\epsilon$ 太大导致过多随机探索、收敛慢；$\epsilon$ 太小容易陷入局部最优。常用==衰减策略==：初期 $\epsilon$ 大（多探索），后期逐渐减小（多利用）。

## 核心知识：无模型方法

### Q-Learning (Off-Policy)

$$Q(s,a) \leftarrow Q(s,a) + \alpha \left[r + \gamma \max_{a'} Q(s', a') - Q(s,a)\right]$$

- ==Off-policy==：行为策略和目标策略不同
- 更新使用 $\max_{a'}$（贪心目标策略）
- 可能高估 Q 值

### SARSA (On-Policy)

$$Q(s,a) \leftarrow Q(s,a) + \alpha \left[r + \gamma Q(s', a') - Q(s,a)\right]$$

- ==On-policy==：行为策略和目标策略相同
- 更新使用实际采取的 $a'$
- 更保守，适合安全敏感场景

### Q-Learning vs SARSA 对比

| 特性 | Q-Learning | SARSA |
|------|-----------|-------|
| 策略类型 | Off-policy | On-policy |
| 更新目标 | $\max_{a'} Q(s',a')$ | $Q(s',a')$ 实际动作 |
| 收敛速度 | 通常更快 | 更稳定 |
| 风险偏好 | 乐观（可能走悬崖边） | 保守（避开危险路径） |

## 核心知识：函数近似与策略梯度

### 函数近似

当状态空间过大（连续状态）时，用参数化函数近似 $Q$：

$$Q(s, a; \theta) \approx Q^*(s, a)$$

- 线性近似：$Q(s,a;\theta) = \theta^T \phi(s,a)$
- 非线性（DQN）：使用神经网络

### 策略梯度 (Policy Gradient)

直接参数化策略 $\pi_\theta(a|s)$，通过梯度上升最大化期望回报：

$$\nabla_\theta J(\theta) = \mathbb{E}_{\pi_\theta}\left[\nabla_\theta \log \pi_\theta(a|s) \cdot G_t\right]$$

其中 $G_t = \sum_{k=0}^{\infty} \gamma^k r_{t+k+1}$ 为回报。

> [!tip] REINFORCE 算法
> 最简单的策略梯度方法。使用蒙特卡洛采样估计梯度，方差高但无偏。常用 ==baseline== 减少方差：$G_t - b(s_t)$。

## 方法选择指南

| 场景 | 推荐方法 | 理由 |
|------|----------|------|
| 图像分类 | CNN | 局部特征 + 平移不变性 |
| 序列预测 | RNN / LSTM | 时序依赖建模 |
| 表格数据分类 | MLP / GBDT | MLP 适合大数据，GBDT 适合中小数据 |
| 有限离散状态 RL | Q-Learning / SARSA | 表格方法，收敛有保证 |
| 连续/大状态空间 RL | DQN / Policy Gradient | 函数近似处理维度诅咒 |
| 安全敏感 RL | SARSA | On-policy，行为更保守 |

## 实战洞察

- 深度学习中，==数据量和质量==往往比模型架构更重要
- CNN 的 `kernel_size=3` 配合 `padding='same'` 是最常见的默认配置
- LSTM 在短文本任务中可能不如 Transformer，但在资源受限场景仍有价值
- Q-Learning 在确定性环境中收敛很快，但在随机环境中需要大量 episode
- $\epsilon$-greedy 的衰减策略（如 $\epsilon = \max(0.01, \epsilon \times 0.995)$）兼顾探索与利用
- DQN 中 ==Experience Replay== 和 ==Target Network== 是稳定训练的两大关键技巧
- 策略梯度方法对学习率极敏感，通常需要 $\eta \leq 10^{-3}$
