# 系统动力学

> [!abstract] 概述
> 基于《Business Dynamics》教材，以==存量-流量-反馈回路==为核心框架，通过Vensim仿真建模分析传染病、产品扩散、库存振荡、服务质量等复杂系统的动态行为。

## 核心知识框架

### 基本概念

- **存量（Stock）**: 系统中的积累量（如库存、感染人数、客户数）
- **流量（Flow）**: 存量的变化率（如生产率、感染率、客户流失率）
- **反馈回路**: ==增强回路==（R，正反馈，指数增长/崩溃）+ ==平衡回路==（B，负反馈，趋于目标）
- **浴缸比喻**: 流量改变存量，存量改变世界；==流速比流量更重要==（GDP增速 > GDP总量）
- **周转时间**: 存量/流量 = 周转时间（Costco 1个月库存周转解释低价高利）

### 系统行为模式

| 行为 | 结构原因 | 典型场景 |
|------|----------|----------|
| 指数增长 | 纯增强回路 | 传染病早期扩散 |
| 目标趋近 | 纯平衡回路 | 恒温器、库存补充 |
| S型增长 | 增强+平衡回路 | 产品生命周期 |
| ==振荡== | 平衡回路+时间延迟 | 牛鞭效应、库存波动 |
| 过冲与崩溃 | 增强回路+资源限制+延迟 | 泡沫经济 |

### 延迟的本质

- **物料延迟**: 一阶延迟（指数平滑），响应特征——阶跃响应渐近、斜坡响应滞后、正弦响应衰减+相移
- **信息延迟**: 感知与决策的滞后
- ==延迟是系统振荡的根本原因==：当调整时间 << 延迟时间，系统从收敛→振荡→放大

## 作业与实践要点

### HW1: 传染病SIR模型（个人）
- SI模型：一个增强回路+一个平衡回路；初始感染者必须>0
- 接触频率/传染性影响扩散速度但不影响最终规模
- SIR模型（加入康复）可使传染病消失
- **不现实之处**: 无恢复、无时间延迟、均匀混合假设

### HW3: VCR产品扩散模型（小组）
- Bass扩散模型 + 价格效应 + 装机量反馈
- 校准历史VCR销售数据
- 安装基数越大→产品吸引力越高→正反馈加速扩散

### HW4: 延迟结构分析
- 一阶物料延迟的显式存量-流量结构（非DELAY1函数）
- 美国天然气市场过剩案例
- 阶跃/斜坡/正弦输入下的延迟响应特征

### HW5: 库存振荡与牛鞭效应（小组）
- Widgets Inc. 制造企业的库存/生产不稳定性
- ==放大率106%==：库存调整时间 << 制造周期时间 → 输出从衰减→振荡→放大
- 双回路系统：库存平衡回路 + WIP（在制品）平衡回路
- 关键参数：WIP调整时间、制造周期、安全库存

### HW6: 服务质量动态（小组）
- "Midnight oil"反馈回路：进度压力→加班→每任务时间减少→质量下降
- 浮动目标结构降低反应性但改善服务质量
- 劳动力噪声对绩效的连锁效应

### HW7: 商业洞察论文（个人）
- **核心存量思维**: 企业要找到自己的"核心存量"（沃尔玛的"低成本"、Intel的"芯片速度"）
- **零售选址新视角**: 地铁口 vs 商场——正确的存量 = 人流量 × 停留时间
- **Costco案例**: 1个月库存周转 → 低价+高利并存的秘密

## 与工作实践的关联

> [!tip] 知识迁移
> 1. **牛鞭效应与库存振荡**→ 直接关联罐箱供应链中的订单波动和产能规划
> 2. **存量-流量思维**→ 用于分析[[Q1经营指标]]中在手订单存量与新签/交付流量的关系
> 3. **延迟结构**→ 解释投资决策到产能形成的时间滞后，对应[[固本—制造与HSE]]中的产能建设节奏
> 4. **服务质量动态**→ 加班→质量下降的反馈回路，对应人效管理
