# 区块链与新技术

> [!abstract] 概述
> 系统介绍区块链的==技术原理==（哈希/共识/密钥）、==架构层次==和==产业应用==，从加密货币到智能合约到供应链应用的演进脉络。

## 核心知识框架

### 区块链本质

- ==分布式、不可篡改的账本系统==
- 解决的核心问题：去信任化的多方协作

### 拜占庭将军问题

- 分布式系统中的一致性难题
- 容错定理：==N ≥ 3M + 1==（N个节点可容忍M个恶意节点）
- 共识机制是区块链的核心创新

### 技术原理

| 技术 | 功能 |
|------|------|
| **哈希加密** | 单向不可逆，任意输入→固定长度输出，微小改动→完全不同 |
| **工作量证明PoW** | 矿工竞争算力，第一个解出难题的获得记账权 |
| **公/私钥系统** | 公钥=地址（公开），私钥=签名（保密），非对称加密 |
| **Merkle树** | 高效验证交易完整性 |

### 架构六层模型

| 层次 | 功能 |
|------|------|
| 数据层 | 区块+链+哈希+时间戳 |
| 网络层 | P2P网络、传播机制 |
| 共识层 | PoW/PoS/DPoS |
| 激励层 | 发行+分配机制 |
| 合约层 | 智能合约、脚本 |
| 应用层 | DApp、产业应用 |

### 区块链演进

| 阶段 | 代表 | 核心应用 |
|------|------|----------|
| **1.0** | Bitcoin | 加密货币、数字支付 |
| **2.0** | Ethereum | ==智能合约==、DeFi |
| **3.0** | 行业应用 | 供应链溯源、跨境支付、数字资产 |

### 供应链应用案例

- **蚂蚁金服跨境汇款**: 利用区块链实现秒级到账+全程可追溯
- **供应链溯源**: 从原材料到成品的全链路信息不可篡改记录
- **Token经济**: 去中心化激励机制的设计
