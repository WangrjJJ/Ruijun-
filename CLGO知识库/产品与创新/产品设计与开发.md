# 产品设计与开发

> [!abstract] 概述
> 以完整的产品开发生命周期为主线，覆盖==用户研究→需求分析→概念生成→系统设计→详细设计→测试==全流程。结合"简约设计"哲学讨论产品设计原则。

## 核心知识框架

### 产品开发流程

```
市场调研 → 客户需求 → 工程规格 → 概念生成 → 系统级设计 → 详细设计 → 测试 → 上市
```

### 简约设计三原则

> [!important] Simplicity设计哲学
> 1. **Delete（删除）**: 去掉非核心功能，聚焦主流用户70%的核心需求
> 2. **Hierarchize（分层）**: 高频功能前置，低频功能隐藏
> 3. **Transfer（转移）**: 将复杂度转移给技术后端或其他载体

- **案例**: 小米电视遥控器——从==72个按键精简到11个==
- **核心理念**: 为主流用户设计（70%），而非专家用户或边缘用户

### 用户研究方法

- 微信问卷调查（105份有效问卷）
- 交叉分析（年龄×使用场景×功能偏好）
- 竞品分析（倍轻松、SKG、Beurer等5家）

## 作业与实践要点

### 小组项目: "懂你的球"——治疗型U形枕

- **市场背景**: 中国==70%人口处于亚健康==状态，白领亚健康率76%
- **用户研究**: 105人微信问卷 → 核心需求：热疗+高频振动，场景：驾车/办公
- **竞品分析**: 5家竞品对比（功能/价格/便携性/材质）
- **产品定义**: 热疗+高频振动的便携式颈腰治疗器
- **工程规格**: 从客户需求转化为可测量的技术参数
- **设计输出**: 概念生成、系统级设计、详细CAD、BOM清单
- **预算**: 5000元原型制作
