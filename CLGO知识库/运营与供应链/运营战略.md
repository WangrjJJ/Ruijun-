# 运营战略

> [!abstract] 概述
> 以案例研究为驱动，从运营战略视角分析企业如何通过==产品、模式、服务==三位一体实现竞争优势。课程覆盖零售、供应链可持续、餐饮精益等多元场景。

## 核心知识框架

### 运营战略的本质

运营战略的本质是==供需匹配==——识别客户真正的需求，然后组织高效的供给。无论何种行业，核心评价维度始终是 **Q（质量）、C（成本）、D（交付）、X（灵活性/其他）**。

### 关键理论工具

| 工具/模型 | 核心思想 | 应用场景 |
|-----------|----------|----------|
| Porter竞争战略 | 运营有效性 ≠ 战略定位；战略需要取舍和活动体系适配 | McDonald's案例 |
| 产品-过程矩阵 | Job Shop → 批量 → 流水线 → 连续流 | 超级工厂对比 |
| 变革管理公式 | 变革动力 = 推动力 - 阻碍力 > 0 | 沃尔玛绿色供应链 |
| 引力弹弓理论 | 借助外力（政策/技术趋势）推进战略目标 | 新能源替代传统燃油车 |
| TCO全生命周期成本 | 不能只看一次性采购成本，要看整个生命周期总成本 | 设备选型、人力投资 |

## 作业与实践要点

### 作业1: 名创优品——零售业运营战略

- **核心论点**: 新零售的本质不是线上+线下，而是==质优价廉==
- **质优策略**: 全球精选800+供应商 + 产品体验官制度（内部+社会招募5000人）
- **价廉策略**: "短路径"（删减中间环节，两点一线）+ "大批量"（买断制海量订单摊薄成本），毛利仅8%
- **运营模式**: 直营10% + 合作15% + 加盟75%；投资风险最小化（严格筛选）+ 运营全面托管（管理/库存/账目）
- **金融创新**: 分利宝等内部金融平台实现资金闭环，是快速扩张的隐藏引擎

### 作业2: 沃尔玛——绿色供应链战略

- **核心框架**: 从推动力与阻碍力两个维度系统分析绿色供应链
- **四大推动力**: ①自身效率提升需求 ②政府/贸易壁垒强制 ③市场环保溢价 ④股东个人追求
- **四大阻碍力**: ①财务负效应 ②企业间缺乏信任 ③技术知识欠缺 ④标准不完备
- **破局策略**: 优先寻找=="引力弹弓"==（借力新能源/数字化等外部趋势），找到与现有体系的==协同点==（运营科学既降本又环保）
- **关键洞察**: 伟大公司起源于疯狂想法，但规模运营需要科学和理性

### 作业3: 西贝莜面村——餐饮业精益运营

- **Q/C/D/X在服务业的应用**:
  - **质量**: "红冰箱"策略（缺陷菜品暂存分析）= 制造业的"红盒子"；服务员主动询问 = EWS早期预警系统
  - **交付**: ==沙漏机制==（25分钟限时上菜）→ 由外而内的管理，倒逼组织流程优化
  - **成本**: 高于市场的薪水策略（给6000换7000贡献 vs 给4000只得2000贡献）
- **精益标准化**: "西贝赛场"绩效体系——2000+工作点纳入考评，=="盯住过程要结果"==
- **精益化实践**: 缩减店面（300㎡→100㎡）、去明火（减少装修周期多赚2月）、标准化器具
- **核心结论**: 核心产品和指标需标准化，非核心流程可灵活化，最终以结果为导向
