# BCG战略咨询框架

> [!abstract] 概述
> 汇集波士顿咨询（BCG）和麦肯锡（McKinsey）的==真实咨询项目报告==，涵盖电信战略、钢铁行业、IT评估、快消营销体系和集团战略等领域的方法论与案例。

## 核心知识框架

### 一、BCG/波士顿咨询项目

#### 1. 中国网通战略咨询

- **背景**: 中国电信行业重组期的战略定位
- **方法**: 行业分析→竞争格局→业务组合→战略选择
- **框架**: BCG经典的==增长-份额矩阵（BCG Matrix）==在电信行业的应用

#### 2. 钢铁企业咨询

- **主题**: 钢铁行业产能整合与竞争力提升
- **分析维度**: 成本曲线分析、产能利用率、产品组合优化
- **与罐箱业的关联**: 钢材是罐箱核心原材料，钢铁行业走势直接影响采购策略

#### 3. 汇源IT评估

- **主题**: 快消企业IT系统能力评估与规划
- **方法**: IT能力成熟度评估 → 差距分析 → 信息化路线图
- **关键输出**: IT架构现状、投资优先级、实施路线图

### 二、麦肯锡咨询项目

#### 1. 宝洁营销体系IDS

- **主题**: 宝洁（P&G）==整合分销系统（IDS）==的营销体系设计
- **核心内容**:
  - 渠道架构设计（直销/分销/零售）
  - 经销商管理体系
  - 终端覆盖与执行标准
  - 促销管理流程
- **方法论价值**: 快消行业营销体系的==标杆设计==

#### 2. 德隆集团战略

- **背景**: 多元化产业集团的战略管控
- **分析框架**:
  - 产业整合战略（横向/纵向）
  - 集团管控模式（战略管控/财务管控/运营管控）
  - 业务组合与资源配置
- **系列报告**: 6份PPT，覆盖战略诊断→方案设计→实施路径

#### 3. 国际汽车城

- **主题**: 汽车产业园区的产业规划与招商策略
- **方法**: 产业链分析→价值链定位→招商方案

#### 4. 康佳战略

- **主题**: 家电企业战略转型
- **系列报告**: 4份PPT，覆盖行业分析、竞争定位、转型路径
- **时代背景**: 中国家电行业从规模扩张到品牌升级的转型期

## 咨询方法论对比

| 维度 | BCG | 麦肯锡 |
|------|-----|--------|
| **经典工具** | BCG矩阵、经验曲线、时基竞争 | 7S模型、MECE原则、金字塔原理 |
| **擅长领域** | 战略组合、成本分析、增长战略 | 组织变革、运营改善、数字化转型 |
| **分析风格** | 量化模型驱动 | 结构化假设驱动 |
| **报告特点** | 数据密集、图表丰富 | 逻辑严密、故事线清晰 |

## 通用咨询框架总结

### 战略咨询标准流程

```
行业分析 → 竞争分析 → 内部能力评估 → 战略选择 → 实施路线图
```

### 关键分析工具

- **外部**: PEST/五力/行业价值链/竞争格局图
- **内部**: 价值链/能力评估/成本结构/组织诊断
- **整合**: SWOT/BCG矩阵/GE矩阵/战略地图
- **财务**: DCF/NPV/敏感性分析/情景规划
