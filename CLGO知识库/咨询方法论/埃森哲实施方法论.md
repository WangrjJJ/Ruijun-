# 埃森哲实施方法论

> [!abstract] 概述
> 基于埃森哲（Accenture）真实项目资料，覆盖==SAP ERP实施==（上海震旦7模块）、==BPO业务流程外包==（苏州供电3阶段）和==战略咨询==（新奥集团）三个完整案例的方法论。

## 核心知识框架

### 一、管理顾问基础能力

- **结构化问题解决**: 假设驱动→数据收集→分析验证→建议
- **咨询项目生命周期**: 售前→启动→诊断→设计→实施→运维交接
- **关键技能**: 结构化思维、客户沟通、演示汇报、团队协作

### 二、SAP ERP实施方法论（上海震旦项目）

#### 实施模块

| 模块 | 全称 | 核心功能 |
|------|------|----------|
| **FI & AM** | 财务会计 & 资产管理 | 总账/应收/应付/固定资产 |
| **CO** | 管理会计 | 成本中心/利润中心/内部订单 |
| **HR** | 人力资源 | 组织管理/人事管理/薪酬/考勤 |
| **PP** | 生产计划 | ==MRP/MPS/产能计划/工单管理== |
| **SM** | 服务管理 | 售后服务/维修工单 |
| **MM** | 物料管理 | ==采购/库存/MRP/供应商管理== |
| **SD** | 销售与分销 | 订单/发货/开票/定价 |

#### 实施方法论（5阶段）

```
项目准备 → 蓝图设计 → 系统实现 → 上线准备 → 上线&支持
```

- **蓝图设计**: AS-IS流程梳理 → TO-BE流程设计 → GAP分析 → 配置方案
- **关键成功因素**: 高层支持、业务流程标准化、数据迁移质量、变革管理

#### PP模块要点（与制造业最相关）

- MRP运算逻辑：需求→BOM展开→库存检查→采购/生产建议
- 工艺路线（Routing）与工作中心（Work Center）配置
- 生产订单生命周期：计划→下达→报工→入库→结算

#### MM模块要点

- 采购流程：请购→询价→采购订单→收货→发票校验
- 库存管理：收发货、库存转移、盘点
- 供应商评估体系

### 三、BPO业务流程外包（苏州供电项目）

#### 项目三阶段

| 阶段 | 内容 | 交付物 |
|------|------|--------|
| **Phase I** | 现状诊断 | AS-IS流程图、问题清单、基线数据 |
| **Phase II** | 方案设计 | TO-BE流程、外包范围、SLA定义、组织设计 |
| **Phase III** | 实施过渡 | 人员转移、系统切换、试运行、绩效监控 |

- **核心逻辑**: 识别非核心业务→标准化流程→外包给专业服务商→SLA管控
- **关键决策**: 哪些流程外包（核心vs非核心）、外包模式、过渡方案

### 四、战略咨询（新奥集团）

- 能源行业战略定位分析
- 业务组合优化与增长路径

## 关键工具与模型

- SAP实施方法论（ASAP / Activate）
- BPO决策矩阵（核心度 × 标准化程度）
- AS-IS/TO-BE流程建模
- GAP分析与需求追踪矩阵
- SLA（服务级别协议）设计
