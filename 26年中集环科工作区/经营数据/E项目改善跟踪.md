# E项目改善跟踪

> [!abstract] 概述
> E项目是2026年罐箱业务降本增效的核心抓手，贯穿采购/设计/工艺三条降本路径。

|------|------|
| 年度销售量 | ==3,150台== |

## 改善类别

### 1. 材料用量优化（预计收益 ==¥1,020/台==）

| 改善项 | 状态 |
|--------|------|
| 封头成型工艺优化 | ✅ 已完成 |
| QN1803材料替代 | ✅ 已完成 |
| 实芯焊丝替代 | ✅ 已完成 |
| 设计结构优化 | ✅ 已完成 |
| 筒体板取消裁边 | 🔄 进行中 |
| 辅材原单位下降 | 🔄 进行中 |

### 2. 采购成本降低

- 标贴、密封胶、铆钉等标准化物料价格谈判
- E项目BOM原单位优化
- 成果横展至其他项目
- 铆接式铭牌框设备供应商现场指导（W13新增）
- 溢流盒确定继续外购，持续关注304材料价格波动

### 3. 产能与效率提升（W13新增）

- 封头车间：单班/双班满产场景规划，明确资源配置（曹建斌）
- 标罐车间：双班生产策划方案制定（王建）
- 溢流盒/爬梯/步道线：外购转自制可行性评估（林爱彬）
- 喷粉材料实际耗用量数据统计验证（李响）
- 喷粉车间燃气能耗数据分析，挖掘优化潜力（王建）
- 工时效率：不同产能场景下的统计分析（王建）

### 4. 成本分析专项（W13新增）

- 3月份E项目已销售订单成本专项分析（张薇/李响）
- 3月份固定费用/变动费用全面分析（张薇）
- 废钢销售价走势合理性判断（张薇）

## W13领导指示（2026-03-27）

> [!important] 张毅总三项要求
> 1. E项目推进措施==持续落地== + ==以点带面==应用到其他订单
> 2. 财务部对3月固定/变动费用全面分析，验证费用管控实施效果
> 3. 重点统计工时/能耗/费用在不同生产规模下的变动规律，==以数据为支撑==辅助决策

## 关键决策点

> [!warning] 需关注
> 1. 筒体板取消裁边和辅材原单位下降尚在进行中，需推动落地
> 2. 单台¥4,000收益是综合计算值，需持续验证
> 3. 成果横展至其他项目是放大收益的关键
> 4. ==（W13新增）==封头/标罐车间已在规划双班满产，为订单回暖做产能准备
> 5. ==（W13新增）==紧急插单导致材料延误+堆场返工，需评估快速反应能力
