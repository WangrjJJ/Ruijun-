# 2026年精益改善地图（PSP版）

> [!abstract] 定位与方法论
> 本文档以 **DBS-PSP（问题解决流程）** 为思考框架，从真问题识别出发，逐层展开现状→理想状态→偏差分析→精益方针，形成可对齐、可讨论、可执行的共识基线。
>
> PSP核心逻辑链：**真问题是什么 → 现状怎样 → 理想状态应是什么 → 偏差在哪里 → 方针（Countermeasure）指向何方 → 具体举措 → 推进机制**。
>
> 数据来源：Q1经营实绩 + 赛意灯塔工厂诊断 + 19方针/绩效合同/ONE推进现状 + 特罐搬迁/阀件并购等重大项目。
>
> 与 [[固本—精益制造]]（车间施策明细）形成 **"地图→行动"** 两层结构。

## 一、真问题识别（PSP Step 1-2）

> PSP原则：不被症状牵着走。区分"问题（Problem）"与"症状（Symptom）"。

### 1.1 症状层：管理层最容易看到的

| 症状 | 数据 | 直观反应 |
|------|------|----------|
| 收入缺口巨大 | Q1营收55,706万(-9% YoY)，预算达成18%，全年缺口~11.5亿 | "要多签单" |
| 利润率极薄 | 1-2月罐箱净利-1,809万，3月才扭亏，标罐亏损1,581万 | "要降成本" |
| 产能利用率奇低 | 38%，损失约6,000万毛利 | "要提效率" |
| 标罐核心产品失速 | 预算达成仅33.8%，同比-26.9% | "标罐出问题了" |
| Q1评为"乏善可陈" | 总经理对公司整体绩效的定性 | "工作没做到位" |

### 1.2 根因层：症状背后的本质

从赛意诊断 + Q1经营数据 + 现场观察中提炼，追问三个"为什么"：

```
为什么产能利用率只有38%？
├── 排产不均衡 ← 主计划与车间计划割裂 ← ETO模式下设计变更频繁
├── 瓶颈工序卡住 ← 涂装/焊接OEE未知 ← 设备OEE基线缺失
└── 缺料等工 ← 采购Leadtime波动 ← MRP人工跟催、无拉动信号

为什么标罐亏损1,581万而特罐增长44%？
├── 标罐是"类大宗商品"，箱价下行（签单价USD 1.38万），成本竞争力不足
└── 特罐定制化有溢价空间，但多品种小批量的效率损失未量化

为什么"乏善可陈"？
├── 指标层：收入/利润/产能三大指标都没达标
├── 系统层：KPI分解到部门但未到班组/个人，无月度对标回顾机制
└── 能力层：标准作业未固化、隐性知识依赖、改善文化靠摊派
```

### 1.3 真问题陈述

> [!danger] 真问题（Problem Statement）
> **中集环科是一家ETO/MTO离散制造企业，订单模式决定了"边设计边生产"是常态。但公司当前的管理体系缺乏端到端的可视化、标准化的作业基线、以及月度对标闭环机制，导致三个后果：**
> 1. **有订单吃不下**：产能利用率38%，不是因为没活干，是因为排产靠人工、瓶颈不可见、异常靠电话
> 2. **吃下去不赚钱**：标罐成本竞争力不足、特罐效率损失大、材料/辅材/能耗缺乏工位级管控
> 3. **改善不可持续**：五个车间各自为政、提案靠摊派、知识在老师傅脑中、KPI有分解无对标
>
> **这不是"市场需求不足"的危机，是"运营能力跟不上业务复杂度"的危机。**

## 二、现状描述（PSP Step 3：厘清现状）

> 只有深刻理解"我们是什么样的制造模式"，才能判断"精益从哪里入手"。

### 2.1 业务模式：ETO + MTO 混合离散制造

| 产品线 | 制造模式 | 特征 | 对精益的约束 |
|--------|----------|------|-------------|
| **标罐** | MTO为主 | 有库存组件，批量相对大，标准化程度较高 | 可以做标准作业、节拍化，SMED重点在换型 |
| **特罐** | ETO纯定制 | 边设计边生产，BOM 3-4级，设计变更频繁，多品种小批量 | 标准作业要做"模块化"，VSM要覆盖设计-生产全流程 |
| **碳罐** | MTO+ETO混合 | 粉末涂装工艺，多能工需求高 | 适合做流动制造试点 |
| **气体罐** | MTO | 焊接密集，气体消耗大 | 辅材管控是降本重点 |
| **小件/封头** | MTS+部分ETO | 品种多、批量小、自制率在提升 | 最适合实践单元化/流动制造 |
| **医疗（MR）** | 独立产品线 | 高端医疗设备配套，客户认证严格 | 品质管理体系需独立于罐箱 |

**核心特征**（赛意诊断 + 实际验证）：

```
订单模式：ETO按单设计 + MTO按单生产，项目型制造
          ├── 接单后设计、采购、加工并行 → OTD周期长
          ├── 边设计边生产 → 设计变更频繁（供应链敏捷度低）
          └── 标罐有库存组件，特罐纯定制化

计划特性：依赖人工调度
          ├── 主计划与车间计划割裂 → 排产不精准
          ├── MRP下达采购，人工跟催送货 → 物料齐套性差
          └── 动态产能反馈不及时 → 销售端不知能承诺什么交期

制造特性：手工派工 + 领料制
          ├── 人员资质校验、物料防错能力不足
          ├── 生产异常靠现场/电话沟通
          └── 质量控制依赖人工，数据分散，无法追溯

产品特性：结构复杂，缺结构化工艺
          ├── BOM 3-4层，设计变更频繁
          ├── EBOM→PBOM关联不足
          ├── 以二维图纸为主，无三维模型和结构化工艺
          └── 关键工艺参数在"老师傅"脑中
```

### 2.2 客户与市场特征

| 维度 | 现状 | 变化趋势 |
|------|------|----------|
| **客户集中度** | 前十大客户长期合作，运营商+租赁商占签单88.7% | 稳定，但客户比价能力极强 |
| **区域分布** | 东亚、大洋洲、西欧为主，外销占比93.8% | 汇率风险敞口大 |
| **订单特征** | Q1新签6,846台(+204% YoY)，但签单价下行（标罐USD 1.38万→在手均价USD 1.31万） | 量增价跌，利润空间被压缩 |
| **交期要求** | 标罐目标7天→5天，非标45天 | 竞争对手也在缩短交期 |
| **质量敏感度** | 外观<90分、错漏装率居高 | 客户对"看起来好不好"比"能用多久"更敏感 |

> [!info] 关键趋势
> 全球罐箱产量连续3年下降（ITCO 4.2万台/2024），但Q1订单+204%说明**结构性机会存在**——问题不是需求不够，是交付能力抢不到这个需求。

### 2.3 供应链现状

| 物料类别 | 供应特征 | 痛点 |
|----------|----------|------|
| **不锈钢板材** | 核心物料，品种规格多（15种板材），采购策略已在MILP模型中优化 | 模型输出与执行之间仍有断点，采购Leadtime波动 |
| **阀件** | 核心供应商之一即将被并购（垂直整合） | 并购后整合期的供应稳定性不确定 |
| **封头** | 135OVC封头已全自制，外销封头预算达成75.2% | 自制能力在提升 |
| **其他外购件** | 供应商数量多，管理复杂度高 | MRP下达后人工跟催，物流跟踪不完善 |
| **辅材/耗材** | 油漆、粉末、焊接气体 | 缺乏工位级消耗管控，降本靠"目标分解"而非"过程管控" |

**供应链的三层问题**：
1. **计划层**：MRP运算→人工维度跟催，无自动拉动信号
2. **执行层**：来料检验、入库、配送仍以人工为主
3. **供应商层**：TOP5供应商帮扶刚开始，供应商Kaizen未启动

### 2.4 制造现场现状（基于已有信息，待现场数据补充）

| 车间 | 关键工艺 | 已知痛点 | 待补充数据 |
|------|----------|----------|-----------|
| **标罐** | 自动焊/机器人、涂装 | 预算达成33.8%，亏损1,581万，外观<90分 | OEE基线、瓶颈工序、换型时间 |
| **特罐** | 防波板/CVC、混气喷涂 | 增长+44.4%，但多品种小批量效率损失 | 换模时间、工时偏差率 |
| **碳罐** | 粉末涂装 | 多能工占比目标40%（年末） | 当前实际多能工比例 |
| **气体罐** | 焊接 | 焊接气体消耗 | 气体消耗原单位 |
| **小件** | 机加工+装配 | 10种以上部件整合进行中 | 工序间在制品数量 |

**已知的共性断层**：
- **工艺-生产断裂**（赛意P11+P12验证）：EBOM→PBOM关联不足，设计变更传导慢
- **设备OEE未量化**（地图M7）：无法识别瓶颈设备，损失根因不明
- **标准作业覆盖率低**：关键工艺参数在老师傅脑中
- **五个车间ONE推进各自为政**：改善成果无法复制

### 2.5 管理体系现状

| 管理维度 | 现状 | 成熟度评估 |
|----------|------|:---:|
| **方针管理** | 年度19方针→各部门绩效合同，KPI v3.0双轨体系已上线 | ⭐⭐⭐（分解有，对标弱） |
| **Daily Management** | 有班组晨会雏形，无标准化三层会议机制 | ⭐（仅雏形） |
| **月度回顾** | 月度经营分析会，财务→各业务线汇报 | ⭐⭐（格式传统，缺SDQCG议程） |
| **问题解决** | TBP 8步法有一定基础，A3报告未制度化 | ⭐⭐ |
| **改善提案** | ≥15件/季度摊派，非员工自驱 | ⭐（有量无质） |
| **人才培养** | 多能工/星级员工/内训师，有目标但缺TWI方法 | ⭐⭐ |
| **精益推进机构** | 无公司级DBSO，精益分散在企管部+各车间 | ⭐（最大缺口） |

## 三、理想状态（PSP Step 4：定义目标状态）

> 基于 DBS 和 ONE 的标杆实践，描绘各维度的理想状态。这个"理想状态"不是五年后的蓝图，而是 **如果我们今天一切做对，12-18个月内应该达到的状态**。

### 3.1 制造维度

| 维度 | 当前状态 | 理想状态（12-18个月） |
|------|----------|----------------------|
| **产能利用率** | 38% | ≥70%，瓶颈工序OEE实时可视 |
| **标准作业** | 老师傅脑中 | 瓶颈工序标准作业覆盖率≥50%，TWI-JI试点运行 |
| **流动/物流** | 推式领料制 | 关键工序推拉结合，标罐实现节拍化生产 |
| **设备管理** | OEE未量化 | 核心设备OEE实时监控，TPM试点1个车间 |
| **质量管理** | 人工检验为主 | 关键工序防错覆盖率≥30%，良品条件制度化 |
| **车间管理** | 各自为政 | 统一精益标准 + 最佳实践月度共享 + 精益成熟度月度评分 |

### 3.2 非制造维度

| 维度 | 当前状态 | 理想状态（12-18个月） |
|------|----------|----------------------|
| **OTD可视化** | 无端到端度量 | 从客户询价→回款全流程周期可度量，TOP3断点已识别 |
| **报价流程** | 多方串联、无SOP | 标准化报价SOP，周期-30%，准确率>95% |
| **工艺-生产协同** | 设计变更传导慢 | ECN标准化流程，变更落地周期-50% |
| **采购执行** | 人工跟催 | 关键物料拉动信号建立，Leadtime波动-30% |
| **间接人效** | 无度量方法 | 事务标准作业基线建立，文干人效可量化 |
| **财务月结** | 多法人多产品线，耗时长 | 月结周期≤D+5工作日 |

### 3.3 管理体系维度

| 维度 | 当前状态 | 理想状态（12-18个月） |
|------|----------|----------------------|
| **方针管理** | 年度分解，季末看一次 | 逐级分解到班组/个人，月度MOR对标（SDQCG议程） |
| **Daily Management** | 雏形 | 三层会议标准化运行（班组晨会→部门日会→工厂日会），KPI小时级暴露 |
| **改善文化** | 摊派≥15件/季度 | 改善提案自驱化，Kaizen发表会每半年1次，有带级激励机制 |
| **精益推进机构** | 无 | DBSO挂牌运作，3-5人核心编制，直接向总经理汇报 |
| **人才育成** | 有目标但无体系 | 黄带/绿带/黑带三级认证，TWI Trainer内部培养 |
| **数字-精益关系** | 各自立项 | 所有数字化项目立项前完成"精益改善→数字化固化"论证 |

### 4.3 偏差优先级矩阵

```
                    对经营的影响大
                         │
    M1 产能利用率        │  M2 标罐竞争力
    M5 搬迁精益设计      │  N1 工艺-生产断裂
    M3 OEE基线           │  N2 OTD可视化
                         │
    ───可控性强──────────┼──────────可控性弱──
                         │
    M4 标准作业          │
    M6 ONE碎片化         │  N7 改善文化
    M7 防错              │  N4 采购Leadtime
    N3 报价流程          │  N6 间接人效
    N5 Daily Management  │
    M8 辅材原单位        │  N8 财务月结
                         │
                    对经营的影响小
```

> **优先级原则**：优先打左上角——影响大 + 可控性强。M5（搬迁精益设计）是**窗口期最短、ROI最高的单一行动**。

### 4.4 关键约束识别

| 约束 | 性质 | 应对策略 |
|------|------|----------|
| ETO模式下设计变更不可避免 | 业务模式约束 | 不追求"消灭变更"，追求"变更传导标准化"（N1） |
| 标罐箱价由全球市场决定 | 外部约束 | 精益聚焦降成本而非涨价（M2 + 采购MILP） |
| 集团考核以ONE为框架 | 体制约束 | DBS作为实施方法论注入，不另起炉灶 |
| 总经理时间有限 | 资源约束 | MOR是唯一必须总经理亲自参加的精益活动 |
| 各车间对"新花样"抵触 | 组织约束 | 先用1-2个速赢Kaizen让结果说话 |

## 九、关键成功因素与风险

### 9.1 必须做对的三件事

1. **总经理亲自站台MOR** — DBS能否落地，唯一最关键变量是总经理是否每月亲自参加MOR并追问"对策是什么"。如果MOR变成企管部的独角戏，整个体系3个月内空心化。

2. **DBSO独立的汇报线** — DBSO必须直接向总经理汇报。挂靠企管部会导致非制造TPI跨部门推动力不足、车间将精益视为"行政任务"。

3. **特罐搬迁的精益设计窗口不能错过** — 施工图定稿前完成VSM未来图+3P分析，是2026年ROI最高的单一精益行动。

### 9.2 风险清单

| 风险 | 概率 | 影响 | 应对 |
|------|:---:|:---:|------|
| PSpace讨论后无法对齐（各部门对"真问题"认知不一致） | 中 | 高 | PSP第一步就是组织跨部门讨论，用数据而非意见说话 |
| 各车间对"又一个新花样"抵触 | 中 | 中 | 先用1-2个速赢Kaizen让结果说话，再推广 |
| DBSO人才难以招聘/内部选拔 | 高 | 中 | 优先从内部有意愿的班组长中选拔培养 |
| 特罐搬迁精益设计与进度冲突 | 中 | 高 | 作为搬迁前置条件写入项目章程 |
| 产能利用率低→"做出来也没订单做"的反对声音 | 低 | 高 | Q1订单+204%是铁证，瓶颈在产线效率而非订单 |
| 集团ONE考核与公司DBS实践口径不一致 | 低 | 低 | ONE为考核框架，DBS为实施方法，课题可作为ONE示范项目申报 |
| 阀件并购交割时间不确定→MFG-06启动延期 | 中 | 中 | 先做好DBS评估模板，交割后即可快速启动 |
