# 马跃一号 — 风险登记册

> 基于DBS Playbook第二章风险矩阵框架，结合阀件精密加工行业特点和Perolo实际，识别并追踪TOP15风险。**风险登记册不只是一张表——PMO每周更新状态，红色风险直接升级委员会。**

---

## 风险评估标准

### 概率评分

| 评分 | 含义 | 判断标准 |
|:--:|------|------|
| 5 | 几乎确定 | >80%概率发生 |
| 4 | 很可能 | 50-80% |
| 3 | 可能 | 20-50% |
| 2 | 不太可能 | 5-20% |
| 1 | 罕见 | <5% |

### 影响评分

| 评分 | 含义 | 判断标准 |
|:--:|------|------|
| 5 | 灾难性 | 损失>¥500万 或 项目延期>3个月 或 重大客户流失 |
| 4 | 严重 | 损失¥100-500万 或 项目延期1-3个月 |
| 3 | 中等 | 损失¥20-100万 或 项目延期2-4周 |
| 2 | 轻微 | 损失¥5-20万 或 项目延期1-2周 |
| 1 | 可忽略 | 损失<¥5万 或 不影响进度 |

### 风险等级矩阵

|  | 影响5 | 影响4 | 影响3 | 影响2 | 影响1 |
|--|:---:|:---:|:---:|:---:|:---:|
| **概率5** | 🔴 25 | 🔴 20 | 🟠 15 | 🟠 10 | 🟡 5 |
| **概率4** | 🔴 20 | 🟠 16 | 🟠 12 | 🟡 8 | 🟡 4 |
| **概率3** | 🟠 15 | 🟠 12 | 🟡 9 | 🟡 6 | 🟢 3 |
| **概率2** | 🟠 10 | 🟡 8 | 🟡 6 | 🟢 4 | 🟢 2 |
| **概率1** | 🟡 5 | 🟢 4 | 🟢 3 | 🟢 2 | 🟢 1 |

🔴 ≥15：立即升级委员会，需专项应对方案
🟠 10-14：PMO重点追踪，工作组组长每周汇报
🟡 5-9：PMO月度监控，有应对预案即可
🟢 1-4：记录在册，按需关注

## TOP15 风险清单（按风险等级排序）

### 🔴 高风险（评分≥15）

#### R1：关键人才流失 — 高技能技师/工程师

| 维度 | 内容 |
|------|------|
| 类别 | 人才与文化 |
| 概率 | 4（很可能） |
| 影响 | 5（灾难性） |
| 评分 | **20** |
| 描述 | 阀件精密加工高度依赖个体技能（数控编程、特殊材料焊接、精密装配）。关键技师/工程师因对并购后不确定性恐慌而离职。此类人才市场稀缺，招聘替代周期6-12个月。Perolo作为制造型精密加工企业，其核心竞争力深嵌在少数高技能技师的经验中。 |
| 预警信号 | ①关键人员开始更新简历 ②请假次数异常增加 ③工作积极性明显下降 ④同行企业开始接触 |
| 预防措施 | ①签约前完成关键人才识别（按"技能不可替代性"而非职级）②签约当天即启动保留协议签署（含留任奖金+里程碑兑现）③派驻GM在Day 1一周内完成所有关键人才的1on1④薪酬不作承诺但明确"不低于现有水平" |
| 应急措施 | 启动紧急招聘+内部技能转移（视频记录+师徒双轨制）+临时外包(如有替代供应商) |
| 责任人 | 组织发展部 |
| PMO追踪 | 周度更新关键人才状态（在职/动摇/已离职） |

#### R2：核心客户因并购不确定性转移订单

| 维度 | 内容 |
|------|------|
| 类别 | 业务连续性 |
| 概率 | 3（可能） |
| 影响 | 5（灾难性） |
| 评分 | **15** |
| 描述 | Perolo的TOP3客户可能占营收50%以上。并购消息传出后，客户担心"新老板会涨价/降质/断供"，主动寻找替代供应商。一旦切换到竞争对手，再赢回难度极大。 |
| 预警信号 | ①客户要求审查合同变更条款 ②客户延迟下单 ③竞争对手开始接触客户 ④客户要求与总经理面谈 |
| 预防措施 | ①Day 1当天营销VP亲自致电TOP20客户 ②提供"业务照常保证函"③一个月内安排CEO/派驻GM拜访TOP5客户 ④明确承诺"供货条款6个月内不变" |
| 应急措施 | 降价/延长账期作为临时维稳手段；派驻客户经理驻场 |
| 责任人 | 包秋婧（营销中心） |
| PMO追踪 | 周度更新TOP客户订单状态和竞争对手动向 |

#### R3：ERP/系统对接失败导致业务中断

| 维度 | 内容 |
|------|------|
| 类别 | IT与数据 |
| 概率 | 3（可能） |
| 影响 | 5（灾难性） |
| 评分 | **15** |
| 描述 | 系统对接（ERP编码映射/科目对照/数据迁移）是并购整合中失败率最高的技术环节。如果在未充分测试的情况下强行切换，可能导致无法下单、无法发货、无法结账——业务停摆。阀件制造涉及大量物料编码、BOM结构和工艺参数，数据迁移复杂度高。 |
| 预警信号 | ①编码映射发现大量不匹配 ②测试环境数据不一致 ③IT团队对原有系统缺乏深入了解 |
| 预防措施 | ①坚持"先用原系统、同步建新系统、测试通过再切换"三步走 ②ERP对接至少提前4周启动准备 ③关键业务节点（下单/发货/结账）保持手动备份流程 ④不允许在100天内做全量系统切换 |
| 应急措施 | 回退到原系统运行；保留纸质/Excel备份流程 |
| 责任人 | 王瑞俊（数字化组） |
| PMO追踪 | 周度更新系统对接测试进度 |

### 🟠 中高风险（评分10-14）

#### R4：质量体系融合不当 — 标准打架或标准滑坡

| 维度 | 内容 |
|------|------|
| 类别 | 运营与质量 |
| 概率 | 4（很可能） |
| 影响 | 3（中等） |
| 评分 | **12** |
| 描述 | 中集和Perolo的质量标准、检验方法、放行流程可能存在显著差异。强硬推行中集标准可能导致"标准过度"（成本飙升但客户不买单），放任原有标准则可能"标准滑坡"（中集品牌受损）。阀件的质量直接影响罐箱的安全性和客户信任。 |
| 预防措施 | ①Day 1起执行"双轨制"——原有标准维持，中集标准作为对比参照 ②第一个月完成双方标准的逐项对比（预计80%相同，20%有实质差异）③对20%的差异逐项判断：中集标准更高→论证升级必要性；原标准更高→保留 |
| 责任人 | 李世社（质量管理部） |
| PMO追踪 | 双周质量数据（PPM/一次合格率/客户投诉）趋势 |

#### R5：派驻GM与原有团队文化冲突

| 维度 | 内容 |
|------|------|
| 类别 | 人才与文化 |
| 概率 | 3（可能） |
| 影响 | 4（严重） |
| 评分 | **12** |
| 描述 | 派驻GM来自中集体系，管理风格、沟通方式、决策节奏可能与原Perolo团队存在显著差异。"大公司来的人"如果姿态不对（指点式而非倾听式），会迅速激起抵触。GM失败是整合失败的最常见原因。 |
| 预防措施 | ①GM选任标准强调"倾听式领导"而非"指令式"②GM在Phase 2（前30天）的首要KPI是"完成所有关键人才1on1+完成3次现场Gemba Walk" 而非"达成多少降本"③企管部在W4组织"GM-团队融合反馈会" |
| 责任人 | 组织发展部（选任）+ 王瑞俊（30天反馈机制） |
| PMO追踪 | 每两周与GM进行1次非正式check-in |

#### R6：供应商整合触发断供

| 维度 | 内容 |
|------|------|
| 类别 | 供应链 |
| 概率 | 3（可能） |
| 影响 | 4（严重） |
| 评分 | **12** |
| 描述 | Perolo现有供应商体系可能与中集的集采策略冲突。急于更换供应商、压价或统一付款条件，可能导致部分供应商主动停止供货。阀件的部分特种材料/锻件供应商本身选择有限，切换周期长。 |
| 预防措施 | ①前30天不更换任何供应商 ②30-100天内仅对"双轨供应商"（双方都在用的）启动集采谈判 ③原Perolo独有供应商至少保留至100天后再评估 |
| 责任人 | 刘建中（采购物流） |
| PMO追踪 | 周度关键供应商交付准时率 |

#### R7：财务管理失控 — 资金漏洞或报表延迟

| 维度 | 内容 |
|------|------|
| 类别 | 财务 |
| 概率 | 2（不太可能） |
| 影响 | 5（灾难性） |
| 评分 | **10** |
| 描述 | 过渡期资金管控盲区：原有财务人员可能因不确定性而懈怠或离职，新增管控流程尚未建立。极端情况下可能出现资金挪用、费用失控、报表严重延迟。作为上市公司，财务并表延迟直接影响季报/年报披露。 |
| 预防措施 | ①Day 1当日完成银行账户授权变更 ②派驻财务经理在Day 1到岗 ③过渡期设置"双签"机制（原财务负责人+派驻财务经理联签>10万支出）④首月财务日报制度 |
| 责任人 | 财务部 |
| PMO追踪 | 周度资金日报完成率 |

### 🟡 中低风险（评分5-9）

#### R8：员工大规模焦虑/谣言传播

| 概率 | 影响 | 评分 | 责任人 |
|:--:|:--:|:--:|------|
| 4 | 2 | **8** | 组织发展部 |

**应对**：①"员工FAQ手册"提前发放 ②Day 1 Town Hall直面敏感问题 ③建立匿名问题收集通道 ④两周内完成全员1on1（中层向下逐级）。组织发展部+企管部联合执行。

#### R9：知识产权泄露或技术流失

| 概率 | 影响 | 评分 | 责任人 |
|:--:|:--:|:--:|------|
| 2 | 4 | **8** | 法务部+技术中心 |

**应对**：①签约前完成IP清单和权属确认 ②关键技术人员签署竞业限制+保密协议 ③技术文档/工艺参数归档备份。Perolo的特殊材料处理工艺和精密加工参数是核心IP。

#### R10：环保/安全历史遗留问题暴露

| 概率 | 影响 | 评分 | 责任人 |
|:--:|:--:|:--:|------|
| 2 | 4 | **8** | HSE+法务部 |

**应对**：①Phase 0完成环保/安全合规排查 ②发现的问题按"是否立即整改"分级 ③储备应急公关方案。精密加工涉及切削液/电镀/热处理等可能有历史隐患的工艺。

#### R11：原股东或有负债暴露

| 概率 | 影响 | 评分 | 责任人 |
|:--:|:--:|:--:|------|
| 2 | 4 | **8** | 法务部+财务部 |

**应对**：①尽调阶段充分排查 ②交易文件中设置赔偿条款 ③保留部分交易对价作为保证金（如可行）。

#### R12：产能爬坡不及预期 — 并购干扰正常生产

| 概率 | 影响 | 评分 | 责任人 |
|:--:|:--:|:--:|------|
| 3 | 2 | **6** | 林爱彬（生产制造） |

**应对**：①Day 1当日确保生产不停线 ②第一个Kaizen直接聚焦产线效率 ③前30天不做任何改变产线布局的动作。

#### R13：汇率/原材料价格波动影响整合期利润

| 概率 | 影响 | 评分 | 责任人 |
|:--:|:--:|:--:|------|
| 3 | 2 | **6** | 财务部 |

**应对**：①整合期不做激进的成本优化承诺 ②在商业计划中预留±10%的成本波动缓冲。钢价/有色金属价格波动直接影响阀件制造成本。

#### R14：过度整合 — "为整合而整合"损耗管理精力

| 概率 | 影响 | 评分 | 责任人 |
|:--:|:--:|:--:|------|
| 3 | 2 | **6** | 王瑞俊（PMO） |

**应对**：①严格执行"做什么/不做什么"清单（百日计划Phase 2的禁止事项）②委员会月度会审查是否出现scope creep ③PMO有责任对"过度整合倾向"喊停。

#### R15：市场/客户对中集"垄断"的负面反应

| 概率 | 影响 | 评分 | 责任人 |
|:--:|:--:|:--:|------|
| 1 | 4 | **5** | 战略发展部+董秘办 |

**应对**：①提前准备反垄断合规论证（中集+Perolo在阀件市场不构成垄断）②如有必要，提前与监管部门沟通 ③对外沟通强调"保障供应链安全"而非"控制市场"。

## 风险升级路径

```
风险状态变化
    │
    ├─ 🔴 红色（≥15）或 🟠橙色（10-14）恶化中
    │       → PMO立即通知委员会副主任
    │       → 48h内召开专项应对会议
    │       → 确定专项负责人和应对方案
    │
    ├─ 🟠 橙色（10-14）正常
    │       → PMO周报标注，工作组组长每周确认
    │       → 连续2周恶化则升级为红色
    │
    └─ 🟡 黄色（5-9）
            → PMO月度监控
            → 连续2月不变或恶化则升级
```

## PMO风险管控节奏

| 频次 | 动作 |
|------|------|
| 每日 | 扫描预警信号（客户投诉/关键人员异常/系统故障） |
| 每周 | 更新TOP15风险状态（红/黄/绿）+ 趋势判断 |
| 每月 | 向委员会提交风险专题报告 + 新增风险登记 + 已关闭风险归档 |
| 里程碑 | 30天/60天/100天三个节点做完整风险评估刷新 |
