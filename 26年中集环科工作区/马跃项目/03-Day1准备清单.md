## Day 1 当日企管部角色

企管部在Day 1是"导演+场记+消防员"：

| 时间 | 行动 |
|------|------|
| 前一天 | 全流程桌面推演（所有关键人员按时间线走一遍） |
| 07:00 | 到场确认：品牌标识/会场/设备/餐饮/礼包全部到位 |
| 08:00 | 收到法务"交割完成"确认 → 发出PMO群"Day 1启动"信号 |
| 09:00 | Town Hall现场：确保议程不跑偏、时间不超、Q&A有人记录 |
| 10:30 | 转场到中高层1on1：确保派驻GM有每个部门的基本信息卡 |
| 全程 | 记录所有员工提出的问题/疑虑（无论是否当场回答） |
| 17:00 | DBSO挂牌：确保总经理或派驻GM亲自揭牌（有仪式感） |
| 18:00 | Day 1总结短会：30分钟，只问三个问题——①今天出了什么岔子？②明天最紧急的三件事？③有没有我们没想到但发生了的？ |
