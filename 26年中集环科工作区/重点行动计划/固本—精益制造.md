# 固本—精益制造

> 2026年重点工作方向 · 详细计划制订中 · 信息整理自绩效合同/商业计划/方针文件

## 一、总体定位

精益制造横跨 **减支-成本领先（方针⑨）** + **固本-制造与HSE（方针⑫⑬⑯）** + **培元-创新与数字化（方针⑰⑱）** 三大方向。当前在 2026 年部门绩效合同中已有分散部署，但缺少统一的精益推进路线图。

### 关键数字

| 指标 | 目标 | 来源 |
|------|------|------|
| 人效提升（一线） | ≥10% | 减支-成本领先 方针⑨ |
| 文干人效提升 | ≥10% | 减支-成本领先 方针⑨ |
| 可比箱型标准工时下降 | 5%~10% | 各车间分解 |
| 制造费用下降 | >10% | 商业计划 |
| 低值易耗/油漆下降 | 5%~10% | 各车间分解 |
| 钢材利用率 | >92% | 固本-工艺降本 |
| 精益培训覆盖率 | 100% | 培元-ONE推进 |
| 星级员工占比 | ≥2%/季度 | 培元-ONE推进 |
| 精益改善提案 | >15件/季度 | 培元-ONE推进 |

## 七、当前缺口（待补充）

1. **缺少统一的精益推进路线图** — 各车间 ONE 推进各自为政，无公司级精益推进办公室
2. **激励机制尚未建立** — 绩效合同中标注为"建立精益激励机制"，目前状态待启动
3. **精益人才培养体系** — 仅有"培训≥4次"，缺少精益带级认证/TWI 等体系化方案
4. **与数字化的协同** — IoT/数据治理如何支撑精益（如 Andon/可视化看板），尚未明确
5. **特罐搬迁中的精益设计** — 新工厂布局/物流/产线设计是否融入精益原则（如 VSM 价值流分析）
