|------|------|
| 衡量指标 | 新业务销售额累计 ≥ ==1,000万== |
| 季度节奏 | Q1>200万 → Q2>400万 → Q3>700万 → Q4>1000万 |
| 责任人 | 朱元春 |

- 加强与科研院所交流
- [ ] 【8月】建成产线、完成样品试制、拓展首家客户
- [ ] 【11月】第二家客户拓展、完成样品交付

## 关键决策点

> [!warning] 需关注
> 1. 国内罐箱市场外销占比93.8%，国内拓展是增量方向但基数小
> 2. 新场景应用（移动储能/液溴）具有首创性，技术风险需评估
> 3. NMR赛道Q1目标200万，需关注进度
