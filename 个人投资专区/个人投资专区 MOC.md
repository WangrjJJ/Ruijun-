## 策略概要

```
4.2 BTC 总持仓
├── A池 35% (1.47 BTC) → 现金流：Sell High双币理财 → 年产¥30万
├── B池 50% (2.10 BTC) → 增币：远期Sell High + 网格 → 年增币≥5%
├── C池 15% (0.63 BTC) → 安全垫：BTC现货冷存储 → 终极防线不动
└── D池 (工资定投USDT) → 机动：月存¥5k-8k → 暴跌时阶梯抄底
```

## 关键指标速览

- **年度现金流目标**: ¥300,000
- **所需最低年化**: ~15.5%
- **A池目标年化**: 18-25%（Sell High短周期）
- **B池目标增币**: ≥0.17 BTC/年
- **C池职能**: 终极安全垫（仅现金流断档/失业时动用）
- **D池抄底阶梯**: $55k→30% / $45k→50% / $40k→全部
