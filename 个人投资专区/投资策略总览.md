## 二、三池分层策略

### A池 - 现金流池（35% | 1.47 BTC）

**目标**：产出30万RMB/年的稳定现金流

**核心策略**：Sell High双币理财
- 行权价：现价 × 1.15（+15%）
- 周期：7-14天（短周期降低暴露）
- 目标年化：18-25%
- 被行权后：用USDT做Buy Low接回BTC（目标价 = 现价 × 0.85~0.90）

**月度提现**：
- 每月初从A池利息中提取 ~$3,625 USDT
- 通过币安P2P/OTC换成RMB（汇率~6.90）
- 月度到账约 ¥25,000

**操作规则**：
- 市场波动率高（IV > 60%）→ 积极开Sell High，权利金更高
- 市场平稳 → 适当缩短周期至3-7天
- 被行权后 → 不追高，耐心用Buy Low在低位接回

### B池 - 增币池（50% | 2.10 BTC）

**目标**：年增币≥5%（≥0.21 BTC）

**策略组合**：
1. **双币理财远期Sell High**（50%｜1.05 BTC）
   - 行权价设更远：现价 × 1.19（+19%以上）
   - 周期：14-30天
   - 不被行权概率高 → 持续赚BTC利息
2. **BTC/USDT现货网格**（20%｜0.42 BTC）
   - 区间：现价 ± 15%
   - 每格利润 0.3%-0.5%
   - 震荡市年化约10-20%
3. **ETH/BTC现货网格**（15%｜0.315 BTC）
   - 交易对：ETH/BTC（quote=BTC，网格利润以BTC结算）
   - 区间：0.017 - 0.035（覆盖当前历史低位+反弹空间）
   - 网格数：30-50格（等差）
   - 逻辑：ETH相对BTC涨→卖ETH收BTC；ETH相对BTC跌→买ETH建仓
   - 底仓：初始投入约一半自动转为ETH
   - 预期年化：3-8% BTC产出（震荡行情）
   - 止损：ETH/BTC跌破0.015 → 暂停网格，剩余ETH换回BTC
   - 止盈：ETH/BTC涨至0.04以上 → 考虑关闭锁定利润
4. **Binance Simple Earn**（15%｜0.315 BTC）
   - 灵活储蓄，年化1-3%
   - 缓冲资金，随时可调配

**年度增币预估**：
- 双币利息：~0.08 BTC（仓位略减）
- BTC/USDT网格利润：~0.05 BTC
- ETH/BTC网格利润：~0.02-0.04 BTC（新增）
- 储蓄利息：~0.005 BTC
- 合计：**~0.16-0.18 BTC（+4-4.3%）**

### C池 - 安全垫池（15% | 0.63 BTC）

**目标**：终极安全网，保住币量底线

**持有形式**：BTC现货（冷钱包或币安灵活存储）

**规则**：
- 平时**绝对不动**，不参与任何策略
- A池现金流严重不足时 → 可部分卖出BTC应急补充
- 失业/收入中断时 → 作为最后防线，分批变现维持生活
- **不再承担抄底职能**（抄底由D池负责）

### D池 - 机动池（工资定投USDT）

**目标**：积累抄底弹药，极端行情低位增仓

**持有形式**：USDT稳定币（币安灵活储蓄）

**资金来源**：每月从工资中拨出 ¥5,000-¥8,000 兑换USDT存入
- 预计3-6个月积累 $3,000-$5,000 USDT

**触发规则**：
- BTC跌破 $55,000（-18%） → 投入D池的30%抄底
- BTC跌破 $45,000（-33%） → 投入D池的50%抄底
- BTC跌破 $40,000（-40%） → D池全部投入 + 评估是否动用C池
- BTC未触发以上条件 → USDT持续积累，赚取灵活储蓄利息（1-3% APY）

## 三、风险控制矩阵

| 风险场景 | 触发条件 | 应对措施 |
|---------|---------|---------|
| BTC暴跌 | 跌幅 >30% | A池暂停提现改Buy Low；D池按阶梯规则抄底 |
| 频繁被行权 | 同月被行权 >2次 | 行权价上调至+20%；缩短周期至3-7天 |
| USDT汇率波动 | 偏离6.90超过5% | 分批提现；关注政策变化 |
| 失业/现金流断档 | 工资收入中断 | A池优先保障提现；D池暂停定投；必要时C池变现应急 |
| 平台风险 | 系统性风险 | 不超过70%资产放币安；C池冷钱包存储 |
| BTC暴涨 | 涨幅 >50% | A池允许部分被行权落袋；B池提高行权价 |
| ETH/BTC持续走弱 | ETH/BTC跌破0.015 | 暂停ETH/BTC网格；剩余ETH换回BTC；仓位并入BTC/USDT网格 |
| ETH/BTC大幅反弹 | ETH/BTC涨至0.04+ | 关闭网格锁定BTC利润；评估是否重开更高区间 |

## 六、核心纪律

1. **不加杠杆** — 永远不用合约/借贷放大仓位
2. **不追涨杀跌** — 被行权就耐心Buy Low接回，不FOMO
3. **月度提现纪律** — 每月固定提取，不因"再等等"而拖延
4. **C池不动纪律** — 除非A池现金流断档或失业，否则C池不变现
5. **D池定投纪律** — 每月工资日固定拨款，不因"行情不错暂时不存"而中断
6. **每周复盘** — 每周一看策略看板，确认本周操作
