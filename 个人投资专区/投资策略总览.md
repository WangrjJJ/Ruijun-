# BTC投资策略总览

> 核心原则：**用收益养现金流，用纪律养币量，用储备养安全感**

## 一、持仓基本面

| 项目 | 数值 |
|------|------|
| 总持仓 | 4.2 BTC |
| 主要平台 | 币安 Binance |
| 主要工具 | 双币理财（Dual Investment） |
| 年度现金流需求 | ≥30万RMB |
| 所需最低年化 | ~15.5%（基于当前市值） |

---

## 二、三池分层策略

### A池 - 现金流池（35% | 1.47 BTC）

**目标**：产出30万RMB/年的稳定现金流

**核心策略**：Sell High双币理财
- 行权价：现价 × 1.15（+15%）
- 周期：7-14天（短周期降低暴露）
- 目标年化：18-25%
- 被行权后：用USDT做Buy Low接回BTC（目标价 = 现价 × 0.85~0.90）

**月度提现**：
- 每月初从A池利息中提取 ~$3,625 USDT
- 通过币安P2P/OTC换成RMB（汇率~6.90）
- 月度到账约 ¥25,000

**操作规则**：
- 市场波动率高（IV > 60%）→ 积极开Sell High，权利金更高
- 市场平稳 → 适当缩短周期至3-7天
- 被行权后 → 不追高，耐心用Buy Low在低位接回

### B池 - 增币池（50% | 2.10 BTC）

**目标**：年增币≥5%（≥0.21 BTC）

**策略组合**：
1. **双币理财远期Sell High**（60%）
   - 行权价设更远：现价 × 1.19（+19%以上）
   - 周期：14-30天
   - 不被行权概率高 → 持续赚BTC利息
2. **BTC现货网格**（25%）
   - 区间：现价 ± 15%
   - 每格利润 0.3%-0.5%
   - 震荡市年化约10-20%
3. **Binance Simple Earn**（15%）
   - 灵活储蓄，年化1-3%
   - 缓冲资金，随时可调配

**年度增币预估**：
- 双币利息：~0.10 BTC
- 网格利润：~0.06 BTC
- 储蓄利息：~0.006 BTC
- 合计：**~0.17 BTC（+4%）**

### C池 - 储备池（15% | 0.63 BTC）

**目标**：纯安全垫，不参与任何策略

**规则**：
- 冷钱包或币安灵活存储
- 仅在BTC暴跌 >40% 时用于抄底补仓
- 或A池现金流严重不足时临时补充
- 平时**绝对不动**

## 三、风险控制矩阵

| 风险场景 | 触发条件 | 应对措施 |
|---------|---------|---------|
| BTC暴跌 | 跌幅 >30% | A池暂停提现改Buy Low；C池视跌幅启动抄底 |
| 频繁被行权 | 同月被行权 >2次 | 行权价上调至+20%；缩短周期至3-7天 |
| USDT汇率波动 | 偏离6.90超过5% | 分批提现；关注政策变化 |
| 失业/现金流断档 | 工资收入中断 | A池优先保障提现；必要时C池解冻 |
| 平台风险 | 系统性风险 | 不超过70%资产放币安；C池冷钱包存储 |
| BTC暴涨 | 涨幅 >50% | A池允许部分被行权落袋；B池提高行权价 |

## 六、核心纪律

1. **不加杠杆** — 永远不用合约/借贷放大仓位
2. **不追涨杀跌** — 被行权就耐心Buy Low接回，不FOMO
3. **月度提现纪律** — 每月固定提取，不因"再等等"而拖延
4. **C池不动纪律** — 除非BTC跌破$40,000，否则C池不解冻
5. **每周复盘** — 每周一看策略看板，确认本周操作
